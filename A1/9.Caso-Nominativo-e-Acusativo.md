# Caso Nominativo e Acusativo

## Introdução

Os casos gramaticais são uma das características mais distintivas do alemão e representam um conceito novo para falantes de português. Em alemão, os substantivos, artigos, adjetivos e pronomes mudam de forma conforme a sua função na frase. Existem quatro casos em alemão: Nominativo, Acusativo, Dativo e Genitivo.

Nesta lição, vamos focar nos dois primeiros e mais importantes: o **Nominativo** (sujeito) e o **Acusativo** (objeto direto). Dominar estes dois casos é fundamental para construir frases corretas em alemão.

---

## 1. O que são Casos Gramaticais?

### 1.1. Conceito Fundamental

Os casos indicam a **função** de um substantivo na frase:
- **Quem** faz a ação? (Nominativo - sujeito)
- **O que/quem** recebe a ação? (Acusativo - objeto direto)
- **A quem/para quem?** (Dativo - veremos na lição 14)
- **De quem?** (Genitivo - níveis mais avançados)

### 1.2. Como Funciona

Os casos afetam:
1. **Artigos** (der/den, ein/einen)
2. **Pronomes** (ich/mich, er/ihn)
3. **Adjetivos** (em níveis mais avançados)

**Exemplo:**
```
Nominativo (sujeito): Der Mann sieht den Hund.
Acusativo (objeto): Der Mann sieht den Hund.
```

O artigo muda conforme a função:
- "Der Mann" (nominativo) = o homem VÊ
- "den Hund" (acusativo) = o homem vê O CÃO

## 2. Caso Nominativo

### 2.1. Função

O Nominativo indica:
1. **O sujeito da frase** (quem faz a ação)
2. **Complemento do verbo "sein"** (ser/estar) e outros verbos copulativos

### 2.2. Artigos no Nominativo

Esta é a forma básica que já conhecemos:

| Género | Definido | Indefinido | Negativo |
|--------|----------|------------|----------|
| **Masculino** | der | ein | kein |
| **Feminino** | die | eine | keine |
| **Neutro** | das | ein | kein |
| **Plural** | die | — | keine |

### 2.3. Perguntas para Identificar o Nominativo

**Wer?** (Quem?) — para pessoas
**Was?** (O quê?) — para coisas

```
Wer kommt? — Der Mann kommt.
(Quem vem? — O homem vem.)

Was ist das? — Das ist ein Buch.
(O que é isto? — Isto é um livro.)
```

### 2.4. Exemplos de Nominativo

#### Como Sujeito
```
Der Mann arbeitet. (O homem trabalha.)
Die Frau liest. (A mulher lê.)
Das Kind spielt. (A criança brinca.)
Die Kinder spielen. (As crianças brincam.)

Ein Hund bellt. (Um cão ladra.)
Eine Katze schläft. (Uma gata dorme.)
Ein Auto fährt. (Um carro circula.)
```

#### Com o Verbo SEIN
```
Ich bin der Lehrer. (Sou o professor.)
Das ist ein Buch. (Isto é um livro.)
Er ist ein guter Mann. (Ele é um bom homem.)
Sie ist die Chefin. (Ela é a chefe.)
Das sind meine Eltern. (Estes são os meus pais.)
```

#### Com Outros Verbos Copulativos
```
Er wird Arzt. (Ele vai ser médico.) — werden
Sie bleibt eine gute Freundin. (Ela continua a ser uma boa amiga.) — bleiben
Er heißt Peter. (Ele chama-se Peter.) — heißen
```

### 2.5. Pronomes Pessoais no Nominativo

| Pessoa | Pronome | Tradução |
|--------|---------|----------|
| 1ª singular | ich | eu |
| 2ª singular | du | tu |
| 3ª singular (M) | er | ele |
| 3ª singular (F) | sie | ela |
| 3ª singular (N) | es | ele/ela (neutro) |
| 1ª plural | wir | nós |
| 2ª plural | ihr | vós |
| 3ª plural | sie | eles/elas |
| Formal | Sie | você/vocês |

**Exemplos:**
```
Ich lerne Deutsch. (Eu aprendo alemão.)
Du bist nett. (Tu és simpático.)
Er kommt morgen. (Ele vem amanhã.)
Sie arbeitet hier. (Ela trabalha aqui.)
Es regnet. (Está a chover.)
Wir gehen ins Kino. (Vamos ao cinema.)
Ihr seid Studenten. (Vós sois estudantes.)
Sie haben Zeit. (Eles têm tempo.)
```

## 3. Caso Acusativo

### 3.1. Função

O Acusativo indica:
1. **O objeto direto** (quem ou o que recebe a ação diretamente)
2. **Usado após certas preposições** (veremos na lição 10)
3. **Expressões de tempo** (ohne Artikel)
4. **Direção de movimento** (com algumas preposições)

### 3.2. Artigos no Acusativo

**Mudança crucial:** Apenas o **masculino singular** muda!

| Género | Definido | Indefinido | Negativo |
|--------|----------|------------|----------|
| **Masculino** | **den** | **einen** | **keinen** |
| **Feminino** | die | eine | keine |
| **Neutro** | das | ein | kein |
| **Plural** | die | — | keine |

**Observação importante:**
- Feminino, neutro e plural permanecem iguais ao nominativo
- Apenas masculino muda: der → den, ein → einen, kein → keinen

### 3.3. Comparação Nominativo vs. Acusativo

| | Nominativo | Acusativo |
|--|------------|-----------|
| **Masculino** | der/ein/kein | **den/einen/keinen** |
| **Feminino** | die/eine/keine | die/eine/keine |
| **Neutro** | das/ein/kein | das/ein/kein |
| **Plural** | die/keine | die/keine |

### 3.4. Perguntas para Identificar o Acusativo

**Wen?** (Quem?) — para pessoas (objeto direto)
**Was?** (O quê?) — para coisas (objeto direto)

```
Wen siehst du? — Ich sehe den Mann.
(Quem vês? — Vejo o homem.)

Was liest du? — Ich lese ein Buch.
(O que lês? — Leio um livro.)
```

### 3.5. Exemplos de Acusativo

#### Masculino (MUDA!)
```
Nominativo: Der Mann ist hier.
Acusativo: Ich sehe den Mann.

Nominativo: Ein Hund bellt.
Acusativo: Ich habe einen Hund.

Nominativo: Das ist kein Lehrer.
Acusativo: Ich kenne keinen Lehrer hier.
```

**Mais exemplos:**
```
Ich sehe den Lehrer. (Vejo o professor.)
Er kennt den Mann. (Ele conhece o homem.)
Wir haben einen Hund. (Temos um cão.)
Sie ruft einen Arzt. (Ela chama um médico.)
Ich brauche einen Stift. (Preciso de uma caneta.)
Er kauft einen Computer. (Ele compra um computador.)
```

#### Feminino (NÃO muda)
```
Nominativo: Die Frau arbeitet.
Acusativo: Ich sehe die Frau.

Nominativo: Eine Katze schläft.
Acusativo: Ich habe eine Katze.
```

**Mais exemplos:**
```
Ich lese die Zeitung. (Leio o jornal.)
Er kennt die Stadt. (Ele conhece a cidade.)
Wir haben eine Wohnung. (Temos um apartamento.)
Sie braucht eine Lampe. (Ela precisa de um candeeiro.)
```

#### Neutro (NÃO muda)
```
Nominativo: Das Kind spielt.
Acusativo: Ich sehe das Kind.

Nominativo: Ein Auto fährt.
Acusativo: Ich kaufe ein Auto.
```

**Mais exemplos:**
```
Ich lese das Buch. (Leio o livro.)
Er hat ein Haus. (Ele tem uma casa.)
Wir sehen das Auto. (Vemos o carro.)
Sie kauft ein Kleid. (Ela compra um vestido.)
```

#### Plural (NÃO muda)
```
Nominativo: Die Kinder spielen.
Acusativo: Ich sehe die Kinder.

Nominativo: Autos fahren schnell.
Acusativo: Ich sehe keine Autos.
```

**Mais exemplos:**
```
Ich habe die Bücher. (Tenho os livros.)
Er kennt die Leute. (Ele conhece as pessoas.)
Wir sehen die Autos. (Vemos os carros.)
```

### 3.6. Pronomes Pessoais no Acusativo

| Nominativo | Acusativo | Tradução |
|------------|-----------|----------|
| ich | mich | me |
| du | dich | te |
| er | ihn | o (ele) |
| sie | sie | a (ela) |
| es | es | o/a (neutro) |
| wir | uns | nos |
| ihr | euch | vos |
| sie | sie | os/as (eles/elas) |
| Sie | Sie | o/a/os/as (formal) |

**Exemplos:**
```
Er sieht mich. (Ele vê-me.)
Ich liebe dich. (Amo-te.)
Siehst du ihn? (Vês-o?)
Wir kennen sie. (Conhecemo-la.)
Ich rufe euch. (Chamo-vos.)
Sie besuchen uns. (Eles visitam-nos.)
```

## 4. Verbos que Regem Acusativo

### 4.1. Verbos Comuns com Acusativo

A maioria dos verbos transitivos em alemão usa acusativo para o objeto direto:

| Verbo | Significado | Exemplo |
|-------|-------------|---------|
| **haben** | ter | Ich habe einen Hund. |
| **sehen** | ver | Ich sehe den Mann. |
| **lesen** | ler | Er liest ein Buch. |
| **kaufen** | comprar | Sie kauft einen Computer. |
| **brauchen** | precisar | Wir brauchen eine Lampe. |
| **essen** | comer | Ich esse einen Apfel. |
| **trinken** | beber | Er trinkt einen Kaffee. |
| **kennen** | conhecer | Ich kenne den Lehrer. |
| **lieben** | amar | Sie liebt ihn. |
| **besuchen** | visitar | Wir besuchen die Stadt. |
| **nehmen** | pegar/tomar | Er nimmt das Buch. |
| **machen** | fazer | Ich mache die Hausaufgaben. |
| **hören** | ouvir | Sie hört die Musik. |
| **finden** | encontrar | Er findet den Schlüssel. |
| **suchen** | procurar | Ich suche einen Stift. |
| **verstehen** | compreender | Wir verstehen den Text. |

### 4.2. Exemplos em Frases

```
Ich habe einen Bruder. (Tenho um irmão.)
Du siehst die Lehrerin. (Vês a professora.)
Er kauft das Auto. (Ele compra o carro.)
Sie liest ein Buch. (Ela lê um livro.)
Wir essen einen Apfel. (Comemos uma maçã.)
Ihr trinkt den Saft. (Bebeis o sumo.)
Sie kennen den Mann. (Eles conhecem o homem.)
Ich brauche einen Computer. (Preciso de um computador.)
```

## 5. Frases com Nominativo E Acusativo

### 5.1. Estrutura Básica

**Sujeito (Nominativo) + Verbo + Objeto Direto (Acusativo)**

```
Der Mann (NOM) sieht den Hund (AKK).
(O homem vê o cão.)

Die Frau (NOM) liest ein Buch (AKK).
(A mulher lê um livro.)

Das Kind (NOM) isst einen Apfel (AKK).
(A criança come uma maçã.)
```

### 5.2. Identificando Nominativo e Acusativo

**Pergunta 1: Quem faz a ação?** → Nominativo (sujeito)
**Pergunta 2: O que/quem recebe a ação?** → Acusativo (objeto)

**Exemplo:**
```
Der Lehrer erklärt den Text.

Wer erklärt? (Quem explica?) → Der Lehrer (Nominativo)
Was erklärt er? (O que explica?) → den Text (Acusativo — masculino!)
```

### 5.3. Exemplos Detalhados

#### Masculino no Acusativo
```
Der Vater kauft einen Wagen. (O pai compra um carro.)
→ Vater = nominativo, Wagen = acusativo (einen!)

Ein Mann sieht den Hund. (Um homem vê o cão.)
→ Mann = nominativo, Hund = acusativo (den!)

Der Schüler macht den Fehler. (O aluno comete o erro.)
→ Schüler = nominativo, Fehler = acusativo (den!)
```

#### Feminino (não muda)
```
Die Mutter liest die Zeitung. (A mãe lê o jornal.)
→ Mutter = nominativo, Zeitung = acusativo (die — igual!)

Ein Kind hat eine Katze. (Uma criança tem uma gata.)
→ Kind = nominativo, Katze = acusativo (eine — igual!)
```

#### Neutro (não muda)
```
Der Lehrer erklärt das Thema. (O professor explica o tema.)
→ Lehrer = nominativo, Thema = acusativo (das — igual!)

Eine Frau kauft ein Haus. (Uma mulher compra uma casa.)
→ Frau = nominativo, Haus = acusativo (ein — igual!)
```

## 6. Pronomes Possessivos nos Casos

### 6.1. Tabela de Pronomes Possessivos

**Nominativo e Acusativo:**

| Pessoa | Base | Masculino NOM | Masculino AKK | Feminino | Neutro | Plural |
|--------|------|---------------|---------------|----------|--------|--------|
| ich | mein | mein | **meinen** | meine | mein | meine |
| du | dein | dein | **deinen** | deine | dein | deine |
| er/es | sein | sein | **seinen** | seine | sein | seine |
| sie | ihr | ihr | **ihren** | ihre | ihr | ihre |
| wir | unser | unser | **unseren** | unsere | unser | unsere |
| ihr | euer | euer | **euren** | eure | euer | eure |
| sie/Sie | ihr/Ihr | ihr/Ihr | **ihren/Ihren** | ihre/Ihre | ihr/Ihr | ihre/Ihre |

**Nota:** Apenas o masculino singular muda no acusativo!

### 6.2. Exemplos com Possessivos

```
Nominativo: Mein Bruder kommt. (O meu irmão vem.)
Acusativo: Ich sehe meinen Bruder. (Vejo o meu irmão.)

Nominativo: Dein Hund ist groß. (O teu cão é grande.)
Acusativo: Ich mag deinen Hund. (Gosto do teu cão.)

Nominativo: Sein Vater arbeitet. (O pai dele trabalha.)
Acusativo: Er besucht seinen Vater. (Ele visita o pai dele.)

Nominativo: Ihre Katze schläft. (A gata dela dorme.)
Acusativo: Sie liebt ihre Katze. (Ela ama a gata dela.) — feminino, não muda!

Nominativo: Unser Auto ist neu. (O nosso carro é novo.)
Acusativo: Wir verkaufen unser Auto. (Vendemos o nosso carro.) — neutro, não muda!
```

## 7. Diálogos Práticos

### Diálogo 1: Na Escola
```
A: Siehst du den Lehrer?
   (Vês o professor?)

B: Ja, ich sehe ihn. Er erklärt das Thema.
   (Sim, vejo-o. Ele explica o tema.)

A: Verstehst du den Text?
   (Compreendes o texto?)

B: Nein, ich verstehe ihn nicht.
   (Não, não o compreendo.)
```

### Diálogo 2: Compras
```
A: Was kaufst du?
   (O que compras?)

B: Ich kaufe einen Computer und eine Lampe.
   (Compro um computador e um candeeiro.)

A: Brauchst du auch einen Drucker?
   (Também precisas de uma impressora?)

B: Nein, ich habe schon einen Drucker.
   (Não, já tenho uma impressora.)
```

### Diálogo 3: Família
```
A: Hast du Geschwister?
   (Tens irmãos?)

B: Ja, ich habe einen Bruder und eine Schwester.
   (Sim, tenho um irmão e uma irmã.)

A: Besuchst du deinen Bruder oft?
   (Visitas o teu irmão frequentemente?)

B: Ja, ich besuche ihn jeden Monat.
   (Sim, visito-o todos os meses.)
```

## 10. Erros Comuns e Como Evitá-los

### Erro 1: Não mudar o masculino no acusativo
**Incorreto:** *Ich sehe der Mann.*
**Correto:** Ich sehe den Mann.

**Solução:** Memorize: masculino singular SEMPRE muda no acusativo (der→den, ein→einen).

### Erro 2: Mudar feminino ou neutro
**Incorreto:** *Ich lese den Buch.* (neutro)
**Correto:** Ich lese das Buch.

**Solução:** Apenas masculino muda! Feminino e neutro ficam iguais.

### Erro 3: Esquecer de mudar os possessivos
**Incorreto:** *Ich sehe mein Bruder.*
**Correto:** Ich sehe meinen Bruder.

**Solução:** Possessivos também mudam no acusativo masculino (mein→meinen).

### Erro 4: Confundir nominativo e acusativo após "sein"
**Incorreto:** *Das ist einen Mann.*
**Correto:** Das ist ein Mann.

**Solução:** Após "sein", sempre use nominativo (não há objeto direto).

### Erro 5: Usar acusativo para o sujeito
**Incorreto:** *Den Mann kommt.*
**Correto:** Der Mann kommt.

**Solução:** O sujeito (quem faz a ação) está SEMPRE no nominativo.

## 11. Dicas de Memorização

### 11.1. Regra de Ouro

**Apenas o MASCULINO SINGULAR muda no acusativo:**
- der → den
- ein → einen
- mein → meinen
- kein → keinen

**Todo o resto permanece igual!**

### 11.2. Método das Perguntas

- **Wer?** (Quem?) → Nominativo
- **Wen?** (Quem?) → Acusativo (para pessoas)
- **Was?** (O quê?) → Nominativo ou Acusativo (conforme contexto)

### 11.3. Flashcards

Crie cartões com frases e pratique identificar os casos:
```
Frente: Der Mann sieht ___ Hund.
Verso: den (masculino, acusativo)
```

### 11.4. Código de Cores

Use cores diferentes para casos:
- **Azul** para Nominativo
- **Vermelho** para Acusativo

## 12. Resumo da Lição

### Pontos-Chave:

1. **Nominativo** = sujeito (quem faz a ação)
   - Pergunta: Wer? Was?
   - Artigos: der/die/das/die

2. **Acusativo** = objeto direto (quem/o que recebe a ação)
   - Pergunta: Wen? Was?
   - Artigos: **den**/die/das/die (apenas masculino muda!)

3. **Mudança crucial:** der → den, ein → einen (masculino singular)

4. **Feminino, neutro e plural:** permanecem iguais nos dois casos

5. **Pronomes:** ich→mich, du→dich, er→ihn, sie→sie, es→es, etc.

6. **Possessivos:** também mudam no acusativo masculino (mein→meinen)
