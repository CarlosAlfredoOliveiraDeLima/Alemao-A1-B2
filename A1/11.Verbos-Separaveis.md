# Verbos Separáveis (trennbare Verben)

## Introdução

Os verbos separáveis (trennbare Verben) são uma característica única e fascinante da língua alemã. Estes verbos são compostos por um prefixo e um verbo base, e numa frase principal no presente ou imperativo, o prefixo separa-se do verbo e vai para o final da frase.

Este conceito não existe em português, o que o torna inicialmente desafiante, mas com prática torna-se natural. Os verbos separáveis são extremamente comuns no alemão quotidiano, por isso dominar este tópico é essencial para comunicar eficazmente.

---

## 1. O que são Verbos Separáveis?

### 1.1. Conceito Fundamental

Um verbo separável é formado por:
**Prefixo separável + Verbo base**

Exemplos:
```
an + kommen = ankommen (chegar)
auf + stehen = aufstehen (levantar-se)
ein + kaufen = einkaufen (fazer compras)
mit + kommen = mitkommen (vir junto)
```

### 1.2. Como Funcionam

**No infinitivo:** o prefixo está junto ao verbo
```
ankommen, aufstehen, einkaufen
```

**Numa frase (presente):** o prefixo separa-se e vai para o final
```
Ich komme um 8 Uhr an. (Chego às 8 horas.)
Er steht früh auf. (Ele levanta-se cedo.)
Wir kaufen im Supermarkt ein. (Fazemos compras no supermercado.)
```

### 1.3. Estrutura da Frase

**Regra de ouro:** O verbo conjugado fica na **posição 2**, o prefixo vai para o **final**.

```
Posição 1 | Posição 2 (verbo) | Meio da frase | Final (prefixo)
----------|-------------------|---------------|----------------
Ich       | komme             | um 8 Uhr      | an.
Er        | steht             | früh          | auf.
Wir       | kaufen            | im Supermarkt | ein.
```

## 2. Prefixos Separáveis Comuns

### 2.1. Lista dos Principais Prefixos Separáveis

Os seguintes prefixos são **sempre** separáveis:

| Prefixo | Significado Básico | Exemplo |
|---------|-------------------|---------|
| **ab-** | para longe, para baixo | abfahren (partir) |
| **an-** | em direção a, ligar | ankommen (chegar) |
| **auf-** | para cima, abrir | aufstehen (levantar-se) |
| **aus-** | para fora, desligar | ausgehen (sair) |
| **ein-** | para dentro, ligar | einkaufen (fazer compras) |
| **mit-** | junto, com | mitkommen (vir junto) |
| **nach-** | depois, verificar | nachdenken (refletir) |
| **vor-** | antes, demonstrar | vorstellen (apresentar) |
| **weg-** | para longe | weggehen (ir embora) |
| **zu-** | fechado, para | zumachen (fechar) |
| **zurück-** | de volta | zurückkommen (voltar) |
| **zusammen-** | juntos | zusammenarbeiten (trabalhar junto) |

**Nota:** Estes prefixos têm **acento tónico** quando pronunciados.

### 2.2. Como Identificar Verbos Separáveis

**Dica 1:** Prefixos separáveis podem funcionar como palavras independentes (preposições ou advérbios).

```
an = em/junto → ankommen
auf = em cima → aufstehen
ein = dentro → einladen
mit = com → mitkommen
```

**Dica 2:** O acento tónico cai no prefixo (não no verbo base).

```
ANkommen (separável) ≠ beKOMMEN (inseparável)
AUFstehen (separável) ≠ verSTEHEN (inseparável)
```

## 3. Verbos Separáveis Mais Comuns

### 3.1. Verbos com AN-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **ankommen** | chegar | Ich komme um 8 Uhr **an**. |
| **anrufen** | telefonar | Er ruft mich **an**. |
| **anfangen** | começar | Der Film fängt um 20 Uhr **an**. |
| **anmachen** | ligar (luz, aparelho) | Sie macht das Licht **an**. |
| **ansehen** | olhar para | Wir sehen den Film **an**. |
| **anziehen** | vestir | Ich ziehe die Jacke **an**. |

**Exemplos detalhados:**
```
Wann kommst du an? (Quando chegas?)
→ Ich komme um 20 Uhr an. (Chego às 20 horas.)

Rufst du mich an? (Telefonas-me?)
→ Ja, ich rufe dich morgen an. (Sim, telefono-te amanhã.)

Wann fängt der Unterricht an? (Quando começa a aula?)
→ Der Unterricht fängt um 9 Uhr an. (A aula começa às 9 horas.)
```

### 3.2. Verbos com AUF-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **aufstehen** | levantar-se | Ich stehe um 7 Uhr **auf**. |
| **aufmachen** | abrir | Er macht die Tür **auf**. |
| **aufhören** | parar, terminar | Wir hören mit der Arbeit **auf**. |
| **aufräumen** | arrumar | Sie räumt das Zimmer **auf**. |
| **aufpassen** | prestar atenção | Passt **auf**! (Prestem atenção!) |
| **aufwachen** | acordar | Ich wache früh **auf**. |

**Exemplos detalhados:**
```
Wann stehst du auf? (Quando te levantas?)
→ Ich stehe um 6 Uhr auf. (Levanto-me às 6 horas.)

Macht er die Tür auf? (Ele abre a porta?)
→ Ja, er macht die Tür auf. (Sim, ele abre a porta.)

Wann hört die Musik auf? (Quando pára a música?)
→ Die Musik hört um 23 Uhr auf. (A música pára às 23 horas.)
```

### 3.3. Verbos com EIN-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **einkaufen** | fazer compras | Wir kaufen im Supermarkt **ein**. |
| **einladen** | convidar | Ich lade dich **ein**. |
| **einschlafen** | adormecer | Das Kind schläft schnell **ein**. |
| **einsteigen** | entrar (veículo) | Steig **ein**! (Entra!) |
| **einziehen** | mudar-se (para casa) | Wir ziehen im Mai **ein**. |

**Exemplos detalhados:**
```
Wo kaufst du ein? (Onde fazes compras?)
→ Ich kaufe im Supermarkt ein. (Faço compras no supermercado.)

Lädst du mich ein? (Convidas-me?)
→ Ja, ich lade dich zur Party ein. (Sim, convido-te para a festa.)

Wann schläfst du ein? (Quando adormecer?)
→ Ich schlafe um 22 Uhr ein. (Adormeço às 22 horas.)
```

### 3.4. Verbos com AUS-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **ausgehen** | sair (para divertimento) | Wir gehen heute Abend **aus**. |
| **ausmachen** | desligar | Er macht das Licht **aus**. |
| **aussehen** | parecer, ter aparência | Du siehst gut **aus**! |
| **aussteigen** | sair (de veículo) | Ich steige hier **aus**. |
| **ausziehen** | despir, mudar-se (sair de casa) | Sie zieht die Jacke **aus**. |

**Exemplos detalhados:**
```
Gehst du heute aus? (Sais hoje?)
→ Ja, ich gehe mit Freunden aus. (Sim, saio com amigos.)

Machst du das Licht aus? (Desligas a luz?)
→ Ja, ich mache das Licht aus. (Sim, desligo a luz.)

Wie sieht er aus? (Como é que ele se parece?)
→ Er sieht jung aus. (Ele parece jovem.)
```

### 3.5. Verbos com MIT-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **mitkommen** | vir junto, acompanhar | Kommst du **mit**? |
| **mitmachen** | participar | Ich mache **mit**! |
| **mitnehmen** | levar consigo | Nimmst du den Regenschirm **mit**? |
| **mitbringen** | trazer consigo | Bring bitte Wein **mit**! |

**Exemplos detalhados:**
```
Kommst du mit ins Kino? (Vens ao cinema?)
→ Ja, ich komme mit! (Sim, vou!)

Machst du bei dem Projekt mit? (Participas no projeto?)
→ Nein, ich mache nicht mit. (Não, não participo.)

Nimmst du einen Regenschirm mit? (Levas um guarda-chuva?)
→ Ja, ich nehme einen mit. (Sim, levo um.)
```

### 3.6. Verbos com ZURÜCK-

| Infinitivo | Significado | Frase Exemplo |
|------------|-------------|---------------|
| **zurückkommen** | voltar | Wann kommst du **zurück**? |
| **zurückgehen** | voltar (a pé) | Wir gehen nach Hause **zurück**. |
| **zurückrufen** | telefonar de volta | Ich rufe dich später **zurück**. |
| **zurückgeben** | devolver | Er gibt das Buch **zurück**. |

**Exemplos detalhados:**
```
Wann kommst du zurück? (Quando voltas?)
→ Ich komme um 18 Uhr zurück. (Volto às 18 horas.)

Rufst du mich zurück? (Telefonas-me de volta?)
→ Ja, ich rufe dich in 10 Minuten zurück. (Sim, telefono-te daqui a 10 minutos.)
```

## 4. Conjugação de Verbos Separáveis

### 4.1. Presente do Indicativo

O verbo base conjuga-se normalmente, o prefixo vai para o final.

**ankommen (chegar):**
```
ich komme ... an
du kommst ... an
er/sie/es kommt ... an
wir kommen ... an
ihr kommt ... an
sie/Sie kommen ... an
```

**aufstehen (levantar-se):**
```
ich stehe ... auf
du stehst ... auf
er/sie/es steht ... auf
wir stehen ... auf
ihr steht ... auf
sie/Sie stehen ... auf
```

### 4.2. Verbos Irregulares Separáveis

Alguns verbos separáveis têm mudanças vocálicas (como verbos irregulares normais):

**ansehen (olhar para) — e → ie:**
```
ich sehe ... an
du siehst ... an
er/sie/es sieht ... an
wir sehen ... an
ihr seht ... an
sie/Sie sehen ... an
```

**einladen (convidar) — a → ä:**
```
ich lade ... ein
du lädst ... ein
er/sie/es lädt ... ein
wir laden ... ein
ihr ladet ... ein
sie/Sie laden ... ein
```

**anfangen (começar) — a → ä:**
```
ich fange ... an
du fängst ... an
er/sie/es fängt ... an
wir fangen ... an
ihr fangt ... an
sie/Sie fangen ... an
```

## 5. Posição em Diferentes Tipos de Frases

### 5.1. Frases Afirmativas

O prefixo vai para o **final** da frase:

```
Ich stehe um 7 Uhr auf.
Er kauft im Supermarkt ein.
Wir gehen heute Abend aus.
Sie kommt morgen zurück.
```

### 5.2. Frases Interrogativas (Perguntas)

#### Perguntas Sim/Não (verbo primeiro):
```
Kommst du mit? (Vens?)
Steht er früh auf? (Ele levanta-se cedo?)
Kauft ihr heute ein? (Fazeis compras hoje?)
Geht sie aus? (Ela sai?)
```

#### Perguntas com Pronome Interrogativo (W-Fragen):
```
Wann stehst du auf? (Quando te levantas?)
Wo kaufst du ein? (Onde fazes compras?)
Warum kommst du nicht mit? (Por que não vens?)
Wie sieht er aus? (Como é a aparência dele?)
```

### 5.3. Frases Negativas

**Com NICHT:**
```
Ich komme nicht mit. (Não vou.)
Er steht nicht früh auf. (Ele não se levanta cedo.)
Wir gehen heute nicht aus. (Não saímos hoje.)
Sie macht das Licht nicht an. (Ela não liga a luz.)
```

**Com KEIN:**
```
Ich kaufe kein Brot ein. (Não compro pão.)
Er nimmt keinen Regenschirm mit. (Ele não leva guarda-chuva.)
```

### 5.4. Frases Subordinadas (introdução)

Em frases subordinadas, o prefixo **não se separa** (veremos em níveis mais avançados):

```
Frase principal: Ich stehe um 7 Uhr auf.
Frase subordinada: Ich weiß, dass ich um 7 Uhr aufstehe.
(Sei que me levanto às 7 horas.)
```

No nível A1, focamos nas frases principais.

## 6. Verbos Separáveis vs. Inseparáveis

### 6.1. Diferença Fundamental

**Separáveis:** O prefixo pode funcionar sozinho como palavra independente
```
ankommen — "an" pode ser uma preposição
aufstehen — "auf" pode ser uma preposição
```

**Inseparáveis:** O prefixo não pode funcionar sozinho
```
bekommen (receber) — "be-" não é uma palavra
verstehen (compreender) — "ver-" não é uma palavra independente
```

### 6.2. Prefixos Inseparáveis Comuns

Estes prefixos **nunca** se separam:
```
be-, emp-, ent-, er-, ge-, miss-, ver-, zer-
```

**Exemplos:**
```
bekommen → Ich bekomme einen Brief. (Recebo uma carta.)
verstehen → Ich verstehe dich. (Compreendo-te.)
erzählen → Er erzählt eine Geschichte. (Ele conta uma história.)
```

**Nota:** O prefixo fica sempre junto ao verbo!

### 6.3. Comparação

| Separável | Inseparável |
|-----------|-------------|
| **ankommen** — Ich komme an. | **bekommen** — Ich bekomme. |
| **aufstehen** — Er steht auf. | **verstehen** — Er versteht. |
| **einkaufen** — Wir kaufen ein. | **verkaufen** — Wir verkaufen. |
| **mitkommen** — Sie kommt mit. | **empfangen** — Sie empfängt. |

## 7. Diálogos Práticos

### Diálogo 1: Rotina Diária
```
A: Wann stehst du normalerweise auf?
   (Quando te costumas levantar?)

B: Ich stehe um 6 Uhr auf. Und du?
   (Levanto-me às 6 horas. E tu?)

A: Ich stehe um 7 Uhr auf. Gehst du auch früh aus dem Haus?
   (Levanto-me às 7 horas. Também sais cedo de casa?)

B: Ja, ich gehe um 7:30 Uhr aus.
   (Sim, saio às 7h30.)
```

### Diálogo 2: Convite
```
A: Wir gehen heute Abend aus. Kommst du mit?
   (Vamos sair hoje à noite. Vens?)

B: Wohin geht ihr?
   (Para onde vão?)

A: Wir gehen ins Kino. Der Film fängt um 20 Uhr an.
   (Vamos ao cinema. O filme começa às 20 horas.)

B: Super! Ich komme gerne mit. Wann treffen wir uns?
   (Ótimo! Vou com prazer. Quando nos encontramos?)

A: Wir treffen uns um 19:30 Uhr vor dem Kino.
   (Encontramo-nos às 19h30 em frente ao cinema.)
```

### Diálogo 3: Compras
```
A: Kaufst du heute noch ein?
   (Ainda fazes compras hoje?)

B: Ja, ich kaufe im Supermarkt ein. Brauchst du etwas?
   (Sim, faço compras no supermercado. Precisas de algo?)

A: Ja, bringst du bitte Milch mit?
   (Sim, trazes leite, por favor?)

B: Kein Problem. Ich bringe Milch mit.
   (Sem problema. Trago leite.)
```

## 10. Erros Comuns e Como Evitá-los

### Erro 1: Não separar o prefixo
**Incorreto:** *Ich ankomme um 8 Uhr.*
**Correto:** Ich komme um 8 Uhr an.

**Solução:** Lembre-se: em frases principais, o prefixo sempre se separa!

### Erro 2: Colocar o prefixo no lugar errado
**Incorreto:** *Ich komme an um 8 Uhr.*
**Correto:** Ich komme um 8 Uhr an.

**Solução:** O prefixo vai para o **final absoluto** da frase.

### Erro 3: Separar verbos inseparáveis
**Incorreto:** *Ich verstehe nicht die Frage.*
**Correto:** Ich verstehe die Frage nicht.

**Solução:** Verbos com be-, ver-, er-, etc. NUNCA se separam.

### Erro 4: Esquecer de conjugar o verbo base
**Incorreto:** *Er ankomme um 8 Uhr.*
**Correto:** Er kommt um 8 Uhr an.

**Solução:** O verbo base conjuga-se normalmente (er kommt).

### Erro 5: Confundir ordem em perguntas
**Incorreto:** *Wann du stehst auf?*
**Correto:** Wann stehst du auf?

**Solução:** Em perguntas com W-, o verbo fica em 2ª posição, prefixo no final.

## 12. Resumo da Lição

### Pontos-Chave:

1. **Verbos separáveis:** Prefixo + verbo base

2. **Separação:** Em frases principais (presente), o prefixo vai para o final

3. **Estrutura:** Verbo conjugado em posição 2, prefixo no final

4. **Prefixos separáveis comuns:** an-, auf-, aus-, ein-, mit-, nach-, vor-, weg-, zu-, zurück-

5. **Prefixos inseparáveis:** be-, emp-, ent-, er-, ge-, miss-, ver-, zer- (nunca se separam)

6. **Conjugação:** O verbo base conjuga-se normalmente

7. **Acento:** Cai no prefixo separável (ANkommen)
