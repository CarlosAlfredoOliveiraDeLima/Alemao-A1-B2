# Verbos Modais Básicos (können, müssen, wollen, möchten)

## Introdução

Os verbos modais (Modalverben) são um grupo especial de verbos que modificam o significado de outros verbos, expressando capacidade, permissão, necessidade, obrigação, desejo, etc. São extremamente comuns no alemão quotidiano e essenciais para expressar nuances de significado.

Nesta lição, vamos focar nos quatro verbos modais mais importantes para o nível A1: **können** (poder, saber), **müssen** (ter que, dever), **wollen** (querer) e **möchten** (gostaria de/querer cortesmente).

---

## 1. Características dos Verbos Modais

### 1.1. O que são Verbos Modais?

Verbos modais são verbos auxiliares que modificam o significado do verbo principal, indicando:
- **Capacidade:** Ich kann schwimmen. (Sei nadar.)
- **Necessidade:** Ich muss arbeiten. (Tenho que trabalhar.)
- **Vontade:** Ich will schlafen. (Quero dormir.)
- **Desejo cortês:** Ich möchte Kaffee. (Gostaria de café.)

### 1.2. Estrutura com Verbos Modais

**Verbo modal conjugado + ... + Infinitivo (no final)**

```
Ich kann Deutsch sprechen.
(Sei falar alemão.)

Er muss heute arbeiten.
(Ele tem que trabalhar hoje.)

Wir wollen ins Kino gehen.
(Queremos ir ao cinema.)
```

### 1.3. Características Especiais

1. **Conjugação irregular:** Todos os verbos modais são irregulares
2. **Singular:** Mudança de vogal (exceto "sollen")
3. **1ª e 3ª pessoa singular:** Sem terminação
4. **Infinitivo no final:** O verbo principal vai para o fim da frase

## 2. KÖNNEN (poder, saber, conseguir)

### 2.1. Significados

**A. Capacidade/Habilidade (saber fazer algo):**
```
Ich kann schwimmen. (Sei nadar.)
Er kann Deutsch sprechen. (Ele sabe falar alemão.)
Sie kann gut kochen. (Ela sabe cozinhar bem.)
```

**B. Possibilidade (poder, ser possível):**
```
Ich kann morgen kommen. (Posso vir amanhã.)
Wir können das machen. (Podemos fazer isso.)
Kannst du mir helfen? (Podes ajudar-me?)
```

**C. Permissão (poder, ter permissão):**
```
Kann ich hier rauchen? (Posso fumar aqui?)
Kannst du gehen. (Podes ir.)
```

### 2.2. Conjugação de KÖNNEN

| Pessoa | Conjugação | Português |
|--------|------------|-----------|
| **ich** | kann | posso/sei |
| **du** | kannst | podes/sabes |
| **er/sie/es** | kann | pode/sabe |
| **wir** | können | podemos/sabemos |
| **ihr** | könnt | podeis/sabeis |
| **sie/Sie** | können | podem/sabem |

**Observações:**
- Singular: "kann" (sem vogal "ö")
- Plural: "können" (com vogal "ö")
- 1ª e 3ª pessoa singular iguais e sem terminação

### 2.3. Exemplos Detalhados

#### Capacidade/Habilidade
```
Ich kann Gitarre spielen. (Sei tocar guitarra.)
Du kannst gut singen. (Sabes cantar bem.)
Er kann drei Sprachen sprechen. (Ele sabe falar três línguas.)
Sie kann schnell laufen. (Ela consegue correr rápido.)
Wir können Deutsch verstehen. (Conseguimos compreender alemão.)
Ihr könnt das Problem lösen. (Conseguis resolver o problema.)
Sie können Auto fahren. (Eles sabem conduzir.)
```

#### Possibilidade
```
Ich kann heute nicht kommen. (Não posso vir hoje.)
Kannst du morgen? (Podes amanhã?)
Er kann um 8 Uhr da sein. (Ele pode estar lá às 8 horas.)
Wir können später anrufen. (Podemos telefonar mais tarde.)
Könnt ihr mitkommen? (Podeis vir connosco?)
Sie können hier bleiben. (Podem ficar aqui.)
```

#### Permissão
```
Kann ich eine Frage stellen? (Posso fazer uma pergunta?)
Kannst du das Fenster öffnen? (Podes abrir a janela?)
Kann man hier parken? (Pode-se estacionar aqui?)
Können wir hier fotografieren? (Podemos fotografar aqui?)
```

### 2.4. KÖNNEN sem Infinitivo

Em alguns contextos, especialmente com línguas, o infinitivo pode ser omitido:

```
Ich kann Deutsch. (Sei alemão.)
— em vez de: Ich kann Deutsch sprechen.

Er kann Englisch. (Ele sabe inglês.)
Kannst du Französisch? (Sabes francês?)
Sie können das. (Eles conseguem isso.)
```

### 2.5. Perguntas e Negações

```
Kannst du schwimmen? (Sabes nadar?)
→ Ja, ich kann schwimmen. / Nein, ich kann nicht schwimmen.

Kann sie kommen? (Ela pode vir?)
→ Ja, sie kann kommen. / Nein, sie kann nicht kommen.

Könnt ihr das machen? (Conseguis fazer isso?)
→ Ja, wir können das machen. / Nein, wir können das nicht machen.
```

## 3. MÜSSEN (ter que, dever)

### 3.1. Significados

**A. Necessidade/Obrigação (ter que, precisar):**
```
Ich muss arbeiten. (Tenho que trabalhar.)
Er muss lernen. (Ele tem que estudar.)
Wir müssen gehen. (Temos que ir.)
```

**B. Dedução lógica (dever ser):**
```
Das muss teuer sein. (Isso deve ser caro.)
Er muss krank sein. (Ele deve estar doente.)
```

### 3.2. Conjugação de MÜSSEN

| Pessoa | Conjugação | Português |
|--------|------------|-----------|
| **ich** | muss | tenho que |
| **du** | musst | tens que |
| **er/sie/es** | muss | tem que |
| **wir** | müssen | temos que |
| **ihr** | müsst | tendes que |
| **sie/Sie** | müssen | têm que |

**Observações:**
- Singular: "muss" (sem "ü")
- Plural: "müssen" (com "ü")
- Forma semelhante a "können"

### 3.3. Exemplos Detalhados

#### Necessidade/Obrigação
```
Ich muss jetzt gehen. (Tenho que ir agora.)
Du musst das verstehen. (Tens que compreender isso.)
Er muss früh aufstehen. (Ele tem que levantar-se cedo.)
Sie muss heute arbeiten. (Ela tem que trabalhar hoje.)
Wir müssen Deutsch lernen. (Temos que aprender alemão.)
Ihr müsst pünktlich sein. (Tendes que ser pontuais.)
Sie müssen hier unterschreiben. (Têm que assinar aqui.)
```

#### Dedução
```
Das muss richtig sein. (Isso deve estar certo.)
Er muss zu Hause sein. (Ele deve estar em casa.)
Sie muss die Lehrerin sein. (Ela deve ser a professora.)
```

### 3.4. MÜSSEN vs. Não BRAUCHEN (nicht müssen)

**Importante:** A negação de "müssen" não é "nicht müssen", mas sim "**nicht brauchen zu**" ou "**nicht müssen**" (com significado diferente):

```
Ich muss arbeiten. (Tenho que trabalhar.)

Ich muss nicht arbeiten. (Não tenho obrigação de trabalhar.)
— significa: não é necessário

Ich brauche nicht zu arbeiten. (Não preciso de trabalhar.)
— forma mais comum para "não precisar"
```

**Nota:** No nível A1, usamos principalmente "nicht müssen":

```
Du musst nicht kommen. (Não precisas de vir.)
Er muss nicht lernen. (Ele não precisa de estudar.)
Ihr müsst nicht warten. (Não precisais de esperar.)
```

### 3.5. Perguntas e Negações

```
Musst du arbeiten? (Tens que trabalhar?)
→ Ja, ich muss arbeiten. / Nein, ich muss nicht arbeiten.

Muss sie kommen? (Ela tem que vir?)
→ Ja, sie muss kommen. / Nein, sie muss nicht kommen.

Müssen wir das machen? (Temos que fazer isso?)
→ Ja, ihr müsst das machen. / Nein, ihr müsst das nicht machen.
```

## 4. WOLLEN (querer)

### 4.1. Significados

**A. Desejo/Vontade (querer):**
```
Ich will schlafen. (Quero dormir.)
Er will nach Berlin fahren. (Ele quer ir a Berlim.)
Wir wollen ins Kino gehen. (Queremos ir ao cinema.)
```

**B. Intenção (pretender):**
```
Was willst du machen? (O que queres fazer?)
Ich will Arzt werden. (Quero ser médico.)
```

### 4.2. Conjugação de WOLLEN

| Pessoa | Conjugação | Português |
|--------|------------|-----------|
| **ich** | will | quero |
| **du** | willst | queres |
| **er/sie/es** | will | quer |
| **wir** | wollen | queremos |
| **ihr** | wollt | quereis |
| **sie/Sie** | wollen | querem |

**Observações:**
- Singular: "will" (sem "o")
- Plural: "wollen" (com "o")
- Padrão semelhante aos outros modais

### 4.3. Exemplos Detalhados

```
Ich will nach Hause gehen. (Quero ir para casa.)
Du willst Deutsch lernen. (Queres aprender alemão.)
Er will ein Auto kaufen. (Ele quer comprar um carro.)
Sie will Lehrerin werden. (Ela quer ser professora.)
Wir wollen heute ausgehen. (Queremos sair hoje.)
Ihr wollt Pizza essen. (Quereis comer pizza.)
Sie wollen ins Museum gehen. (Eles querem ir ao museu.)
```

### 4.4. WOLLEN sem Infinitivo

Com destinos ou objetos diretos:

```
Ich will nach Berlin. (Quero ir a Berlim.)
— em vez de: Ich will nach Berlin fahren.

Er will ein Bier. (Ele quer uma cerveja.)
Was willst du? (O que queres?)
Wir wollen Pizza. (Queremos pizza.)
```

### 4.5. Nota Cultural

"Wollen" pode soar um pouco direto ou forte. Para ser mais educado, usa-se **möchten** (veremos a seguir).

```
Menos cortês: Ich will Kaffee. (Quero café.)
Mais cortês: Ich möchte Kaffee. (Gostaria de café.)
```

### 4.6. Perguntas e Negações

```
Willst du mitkommen? (Queres vir?)
→ Ja, ich will mitkommen. / Nein, ich will nicht mitkommen.

Will er das kaufen? (Ele quer comprar isso?)
→ Ja, er will das kaufen. / Nein, er will das nicht kaufen.

Wollt ihr bleiben? (Quereis ficar?)
→ Ja, wir wollen bleiben. / Nein, wir wollen nicht bleiben.
```

## 5. MÖCHTEN (gostaria de, querer cortesmente)

### 5.1. Significados

**Desejo cortês (gostaria de):**

"Möchten" é a forma de Konjunktiv II (subjuntivo) de "mögen" (gostar), mas é usado como verbo modal independente, sendo mais educado que "wollen".

```
Ich möchte Kaffee. (Gostaria de café.)
Er möchte nach Berlin fahren. (Ele gostaria de ir a Berlim.)
Wir möchten das Buch kaufen. (Gostaríamos de comprar o livro.)
```

### 5.2. Conjugação de MÖCHTEN

| Pessoa | Conjugação | Português |
|--------|------------|-----------|
| **ich** | möchte | gostaria |
| **du** | möchtest | gostarias |
| **er/sie/es** | möchte | gostaria |
| **wir** | möchten | gostaríamos |
| **ihr** | möchtet | gostaríeis |
| **sie/Sie** | möchten | gostariam |

**Observações:**
- A vogal "ö" mantém-se em todas as formas
- Terminações regulares
- Forma mais educada que "wollen"

### 5.3. Exemplos Detalhados

#### Em Restaurantes/Cafés
```
Ich möchte einen Kaffee. (Gostaria de um café.)
Möchtest du etwas essen? (Gostarias de comer algo?)
Er möchte ein Wasser, bitte. (Ele gostaria de uma água, por favor.)
Wir möchten die Rechnung. (Gostaríamos da conta.)
Was möchten Sie trinken? (O que gostaria de beber?)
```

#### Planos e Desejos
```
Ich möchte nach Berlin fahren. (Gostaria de ir a Berlim.)
Möchtest du ins Kino gehen? (Gostarias de ir ao cinema?)
Sie möchte Deutsch lernen. (Ela gostaria de aprender alemão.)
Wir möchten ein Haus kaufen. (Gostaríamos de comprar uma casa.)
Möchtet ihr mitkommen? (Gostaríeis de vir?)
Sie möchten hier bleiben. (Eles gostariam de ficar aqui.)
```

### 5.4. MÖCHTEN sem Infinitivo

Muito comum com objetos diretos:

```
Ich möchte Kaffee. (Gostaria de café.)
Möchtest du Pizza? (Queres pizza?)
Er möchte einen Tee. (Ele gostaria de um chá.)
Was möchten Sie? (O que deseja?)
Wir möchten zwei Tickets. (Gostaríamos de dois bilhetes.)
```

### 5.5. WOLLEN vs. MÖCHTEN

| WOLLEN (direto) | MÖCHTEN (cortês) |
|-----------------|------------------|
| Ich will Kaffee. | Ich möchte Kaffee. |
| (Quero café.) | (Gostaria de café.) |
| Was willst du? | Was möchtest du? |
| (O que queres?) | (O que gostarias?) |

**Regra prática:** Use "möchten" em situações formais, restaurantes, lojas, ou quando quer ser educado.

### 5.6. Perguntas e Negações

```
Möchtest du Kaffee? (Gostarias de café?)
→ Ja, ich möchte Kaffee. / Nein, ich möchte keinen Kaffee.

Möchten Sie etwas trinken? (Gostaria de beber algo?)
→ Ja, ich möchte etwas trinken. / Nein, danke.

Möchtet ihr mitkommen? (Gostaríeis de vir?)
→ Ja, wir möchten mitkommen. / Nein, wir möchten nicht mitkommen.
```

## 6. Comparação dos Quatro Verbos Modais

### 6.1. Tabela Comparativa Completa

| Pessoa | KÖNNEN | MÜSSEN | WOLLEN | MÖCHTEN |
|--------|--------|--------|--------|---------|
| **ich** | kann | muss | will | möchte |
| **du** | kannst | musst | willst | möchtest |
| **er/sie/es** | kann | muss | will | möchte |
| **wir** | können | müssen | wollen | möchten |
| **ihr** | könnt | müsst | wollt | möchtet |
| **sie/Sie** | können | müssen | wollen | möchten |

### 6.2. Uso em Contexto

```
Ich kann schwimmen. (Sei nadar — capacidade)
Ich muss arbeiten. (Tenho que trabalhar — obrigação)
Ich will schlafen. (Quero dormir — vontade direta)
Ich möchte schlafen. (Gostaria de dormir — vontade cortês)
```

### 6.3. Numa Frase Completa

```
Ich kann heute nicht kommen, weil ich arbeiten muss. Ich will lieber zu Hause bleiben, aber ich möchte dich bald sehen.

(Não posso vir hoje porque tenho que trabalhar. Quero ficar em casa, mas gostaria de ver-te em breve.)
```

## 7. Posição do Infinitivo

### 7.1. Regra Fundamental

O infinitivo vai **sempre para o final** da frase:

```
Ich kann Deutsch sprechen.
Du musst heute arbeiten.
Er will nach Berlin fahren.
Wir möchten Pizza essen.
```

### 7.2. Com Verbos Separáveis

O infinitivo do verbo separável fica junto (não separado):

```
Ich kann früh aufstehen. (Consigo levantar-me cedo.)
— NÃO: Ich kann früh stehen auf.

Er muss heute einkaufen. (Ele tem que fazer compras hoje.)
Wir wollen morgen ausgehen. (Queremos sair amanhã.)
Möchtest du mitkommen? (Gostarias de vir?)
```

### 7.3. Em Perguntas

```
Kannst du schwimmen? (Sabes nadar?)
Musst du arbeiten? (Tens que trabalhar?)
Willst du mitkommen? (Queres vir?)
Möchten Sie etwas trinken? (Gostaria de beber algo?)
```

### 7.4. Com Negação

"Nicht" geralmente vem antes do infinitivo:

```
Ich kann nicht schwimmen. (Não sei nadar.)
Du musst nicht kommen. (Não precisas de vir.)
Er will nicht arbeiten. (Ele não quer trabalhar.)
Wir möchten nicht bleiben. (Não gostaríamos de ficar.)
```

## 8. Diálogos Práticos

### Diálogo 1: No Restaurante
```
Kellner: Guten Tag! Was möchten Sie trinken?
         (Bom dia! O que gostaria de beber?)

Kunde: Ich möchte einen Kaffee, bitte.
       (Gostaria de um café, por favor.)

Kellner: Möchten Sie auch etwas essen?
         (Gostaria também de comer algo?)

Kunde: Ja, ich möchte ein Sandwich.
       (Sim, gostaria de uma sandes.)

Kellner: Sehr gern. Das kommt sofort.
         (Com certeza. Já vem.)
```

### Diálogo 2: Planeando o Fim de Semana
```
A: Was willst du am Wochenende machen?
   (O que queres fazer no fim de semana?)

B: Ich möchte ins Kino gehen. Kannst du mitkommen?
   (Gostaria de ir ao cinema. Podes vir?)

A: Nein, leider kann ich nicht. Ich muss arbeiten.
   (Não, infelizmente não posso. Tenho que trabalhar.)

B: Musst du den ganzen Tag arbeiten?
   (Tens que trabalhar o dia todo?)

A: Ja, ich muss von 9 bis 18 Uhr arbeiten.
   (Sim, tenho que trabalhar das 9 às 18 horas.)

B: Schade! Vielleicht nächste Woche?
   (Pena! Talvez na próxima semana?)

A: Ja, nächste Woche kann ich!
   (Sim, na próxima semana posso!)
```

### Diálogo 3: Habilidades
```
A: Kannst du Gitarre spielen?
   (Sabes tocar guitarra?)

B: Nein, ich kann nicht Gitarre spielen. Aber ich kann Klavier spielen.
   (Não, não sei tocar guitarra. Mas sei tocar piano.)

A: Toll! Wie lange kannst du schon Klavier spielen?
   (Ótimo! Há quanto tempo sabes tocar piano?)

B: Ich kann seit 5 Jahren Klavier spielen.
   (Sei tocar piano há 5 anos.)

A: Ich will auch ein Instrument lernen!
   (Também quero aprender um instrumento!)
```

## 11. Erros Comuns e Como Evitá-los

### Erro 1: Não colocar o infinitivo no final
**Incorreto:** *Ich kann sprechen Deutsch.*
**Correto:** Ich kann Deutsch sprechen.

### Erro 2: Conjugar o verbo principal
**Incorreto:** *Ich muss arbeitest.*
**Correto:** Ich muss arbeiten.

**Nota:** Apenas o verbo modal se conjuga; o verbo principal fica no infinitivo.

### Erro 3: Separar verbos separáveis com modais
**Incorreto:** *Ich kann stehe früh auf.*
**Correto:** Ich kann früh aufstehen.

### Erro 4: Usar "wollen" em situações formais
**Menos adequado:** *Ich will einen Kaffee.* (em restaurante)
**Mais adequado:** Ich möchte einen Kaffee.

### Erro 5: Confundir "nicht müssen" com proibição
**Nicht müssen** = não é necessário
**Nicht dürfen** = não é permitido (veremos em níveis mais avançados)

```
Du musst nicht kommen. (Não precisas de vir — não é obrigatório)
Du darfst nicht rauchen. (Não podes fumar — é proibido)
```

## 12. Resumo da Lição

### Pontos-Chave:

1. **Verbos modais:** Modificam o significado de outros verbos

2. **Quatro principais:**
   - **können** — poder, saber (capacidade)
   - **müssen** — ter que (obrigação)
   - **wollen** — querer (vontade)
   - **möchten** — gostaria de (desejo cortês)

3. **Estrutura:** Verbo modal conjugado + ... + infinitivo (no final)

4. **Conjugação:** Todos irregulares, sem terminação na 1ª e 3ª pessoa singular

5. **Cortesia:** Usar "möchten" em vez de "wollen" em situações formais
