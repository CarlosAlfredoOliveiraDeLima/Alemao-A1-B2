# Frases Interrogativas (W-Fragen e Ja/Nein-Fragen)

## Introdução

Fazer perguntas é uma das competências mais fundamentais em qualquer língua. Em alemão, existem dois tipos principais de perguntas: as **perguntas sim/não** (Ja/Nein-Fragen ou Entscheidungsfragen), onde a resposta é simplesmente "sim" ou "não", e as **perguntas com pronomes interrogativos** (W-Fragen), que pedem informação específica.

Dominar a formação de perguntas é essencial para manter conversas, pedir informações e interagir eficazmente em alemão.

---

## 1. Perguntas Sim/Não (Ja/Nein-Fragen)

### 1.1. Estrutura Básica

Nas perguntas sim/não, o **verbo conjugado vai para a primeira posição** da frase.

**Estrutura:**
```
Verbo + Sujeito + Resto da frase + ?
```

**Comparação:**
```
Afirmativa: Du sprichst Deutsch. (Falas alemão.)
Pergunta: Sprichst du Deutsch? (Falas alemão?)
```

### 1.2. Formação de Perguntas Sim/Não

#### Com Verbos Regulares
```
Afirmativa: Er wohnt in Berlin.
Pergunta: Wohnt er in Berlin?
(Ele mora em Berlim?)

Afirmativa: Sie lernen Deutsch.
Pergunta: Lernen sie Deutsch?
(Eles aprendem alemão?)

Afirmativa: Ihr arbeitet hier.
Pergunta: Arbeitet ihr hier?
(Trabalhais aqui?)
```

#### Com SEIN
```
Afirmativa: Du bist müde.
Pergunta: Bist du müde?
(Estás cansado?)

Afirmativa: Das ist interessant.
Pergunta: Ist das interessant?
(Isso é interessante?)

Afirmativa: Sie sind aus Deutschland.
Pergunta: Sind Sie aus Deutschland?
(Você é da Alemanha?)
```

#### Com HABEN
```
Afirmativa: Er hat Zeit.
Pergunta: Hat er Zeit?
(Ele tem tempo?)

Afirmativa: Ihr habt Hunger.
Pergunta: Habt ihr Hunger?
(Tendes fome?)

Afirmativa: Sie haben Kinder.
Pergunta: Haben Sie Kinder?
(Têm filhos?)
```

#### Com Verbos Modais (introdução)
```
Afirmativa: Du kannst Deutsch.
Pergunta: Kannst du Deutsch?
(Sabes alemão?)

Afirmativa: Er muss arbeiten.
Pergunta: Muss er arbeiten?
(Ele precisa de trabalhar?)
```

#### Com Verbos Separáveis
```
Afirmativa: Du stehst früh auf.
Pergunta: Stehst du früh auf?
(Levantas-te cedo?)

Afirmativa: Sie kommt mit.
Pergunta: Kommt sie mit?
(Ela vem?)

Afirmativa: Ihr kauft heute ein.
Pergunta: Kauft ihr heute ein?
(Fazeis compras hoje?)
```

### 1.3. Respostas a Perguntas Sim/Não

#### Respostas Afirmativas
```
Pergunta: Sprichst du Deutsch?
Resposta: Ja, ich spreche Deutsch.
(Sim, falo alemão.)

Pergunta: Ist das interessant?
Resposta: Ja, das ist interessant.
(Sim, isso é interessante.)
```

#### Respostas Negativas
```
Pergunta: Hast du Zeit?
Resposta: Nein, ich habe keine Zeit.
(Não, não tenho tempo.)

Pergunta: Kommst du mit?
Resposta: Nein, ich komme nicht mit.
(Não, não vou.)
```

#### Resposta DOCH (contradizendo uma pergunta negativa)
```
Pergunta: Sprichst du nicht Deutsch?
Resposta: Doch, ich spreche Deutsch!
(Sim, falo alemão!) — contradiz a negativa

Pergunta: Hast du kein Auto?
Resposta: Doch, ich habe ein Auto!
(Sim, tenho carro!) — contradiz a negativa
```

## 2. Perguntas com Pronomes Interrogativos (W-Fragen)

### 2.1. Estrutura Básica

Nas perguntas com pronomes interrogativos, a estrutura é:

**Estrutura:**
```
Pronome interrogativo + Verbo + Sujeito + Resto da frase + ?
```

**Exemplo:**
```
Was machst du?
(O que fazes?)

Wo wohnst du?
(Onde moras?)

Wann kommst du?
(Quando vens?)
```

### 2.2. Pronomes Interrogativos Principais

| Alemão | Português | Exemplo |
|--------|-----------|---------|
| **wer** | quem (sujeito) | Wer ist das? |
| **wen** | quem (objeto acusativo) | Wen siehst du? |
| **wem** | a quem (dativo) | Wem gibst du das Buch? |
| **wessen** | de quem (genitivo) | Wessen Buch ist das? |
| **was** | o quê | Was machst du? |
| **wo** | onde | Wo wohnst du? |
| **wohin** | para onde | Wohin gehst du? |
| **woher** | de onde | Woher kommst du? |
| **wann** | quando | Wann kommst du? |
| **warum** | por quê | Warum lernst du Deutsch? |
| **wie** | como | Wie heißt du? |
| **wie viel** | quanto (singular) | Wie viel kostet das? |
| **wie viele** | quantos (plural) | Wie viele Kinder hast du? |
| **welcher** | qual (masculino) | Welcher Mann? |
| **welche** | qual (feminino/plural) | Welche Frau? |
| **welches** | qual (neutro) | Welches Buch? |

## 3. WER, WEN, WEM, WESSEN (Quem)

### 3.1. WER (Nominativo - Sujeito)

Pergunta sobre o **sujeito** (quem faz a ação):

```
Wer ist das? (Quem é?)
→ Das ist Peter.

Wer kommt? (Quem vem?)
→ Maria kommt.

Wer arbeitet hier? (Quem trabalha aqui?)
→ Der Lehrer arbeitet hier.

Wer spricht Deutsch? (Quem fala alemão?)
→ Ich spreche Deutsch.
```

### 3.2. WEN (Acusativo - Objeto Direto)

Pergunta sobre o **objeto direto** (quem recebe a ação):

```
Wen siehst du? (Quem vês?)
→ Ich sehe den Lehrer.

Wen liebst du? (Quem amas?)
→ Ich liebe dich.

Wen rufst du an? (A quem telefonas?)
→ Ich rufe meinen Bruder an.

Wen kennt er? (Quem ele conhece?)
→ Er kennt die Frau.
```

### 3.3. WEM (Dativo - Objeto Indireto)

Pergunta sobre o **objeto indireto** (a quem) — veremos mais na lição 14:

```
Wem gibst du das Buch? (A quem dás o livro?)
→ Ich gebe dem Lehrer das Buch.

Wem hilfst du? (A quem ajudas?)
→ Ich helfe meiner Mutter.
```

### 3.4. WESSEN (Genitivo - Posse)

Pergunta sobre **posse** (de quem) — menos comum no A1:

```
Wessen Buch ist das? (De quem é este livro?)
→ Das ist Peters Buch.

Wessen Auto ist das? (De quem é este carro?)
→ Das ist mein Auto.
```

## 4. WAS (O Quê)

### 4.1. Usos Principais

**Perguntar sobre coisas/ações:**

```
Was ist das? (O que é isto?)
→ Das ist ein Buch.

Was machst du? (O que fazes?)
→ Ich lerne Deutsch.

Was kostet das? (Quanto custa isso?)
→ Das kostet 10 Euro.

Was möchtest du? (O que queres?)
→ Ich möchte Kaffee.

Was gibt es? (O que há?)
→ Es gibt Kuchen.

Was bedeutet das? (O que significa isso?)
→ Das bedeutet "Hallo".
```

### 4.2. WAS FÜR (Que Tipo De)

```
Was für ein Auto hast du? (Que tipo de carro tens?)
→ Ich habe einen BMW.

Was für Musik magst du? (Que tipo de música gostas?)
→ Ich mag Rockmusik.
```

## 5. WO, WOHIN, WOHER (Onde)

### 5.1. WO (Onde - Localização)

Pergunta sobre **localização estática**:

```
Wo wohnst du? (Onde moras?)
→ Ich wohne in Berlin.

Wo bist du? (Onde estás?)
→ Ich bin zu Hause.

Wo arbeitest du? (Onde trabalhas?)
→ Ich arbeite im Büro.

Wo ist das Buch? (Onde está o livro?)
→ Das Buch ist auf dem Tisch.

Wo kaufst du ein? (Onde fazes compras?)
→ Ich kaufe im Supermarkt ein.
```

### 5.2. WOHIN (Para Onde - Direção)

Pergunta sobre **direção/destino**:

```
Wohin gehst du? (Para onde vais?)
→ Ich gehe ins Kino.

Wohin fährst du? (Para onde vais de carro/transporte?)
→ Ich fahre nach Berlin.

Wohin reist du? (Para onde viajas?)
→ Ich reise nach Portugal.

Wohin kommst du? (Para onde vens?)
→ Ich komme zum Bahnhof.
```

### 5.3. WOHER (De Onde - Origem)

Pergunta sobre **origem/proveniência**:

```
Woher kommst du? (De onde és? / De onde vens?)
→ Ich komme aus Portugal.

Woher bist du? (De onde és?)
→ Ich bin aus Lissabon.

Woher hast du das? (De onde tens isso?)
→ Ich habe das aus dem Laden.

Woher weißt du das? (De onde sabes isso?)
→ Ich weiß das von meinem Freund.
```

**Dica mnemónica:**
- **WO** = onde (estático)
- **WOhin** = para onde (movimento para)
- **WOher** = de onde (origem)

## 6. WANN (Quando)

### 6.1. Usos

Pergunta sobre **tempo**:

```
Wann kommst du? (Quando vens?)
→ Ich komme um 8 Uhr.

Wann beginnt der Unterricht? (Quando começa a aula?)
→ Der Unterricht beginnt um 9 Uhr.

Wann hast du Geburtstag? (Quando fazes anos?)
→ Ich habe am 15. Mai Geburtstag.

Wann fährt der Zug? (Quando parte o comboio?)
→ Der Zug fährt um 10:30 Uhr.

Wann bist du zu Hause? (Quando estás em casa?)
→ Ich bin um 18 Uhr zu Hause.
```

---

## 7. WARUM (Por Quê)

### 7.1. Usos

Pergunta sobre **razão/causa**:

```
Warum lernst du Deutsch? (Por que aprendes alemão?)
→ Weil ich in Deutschland wohne. (Porque moro na Alemanha.)

Warum bist du müde? (Por que estás cansado?)
→ Weil ich nicht gut geschlafen habe.

Warum kommst du nicht? (Por que não vens?)
→ Weil ich keine Zeit habe.

Warum arbeitest du hier? (Por que trabalhas aqui?)
→ Weil die Arbeit interessant ist.
```

**Nota:** A resposta geralmente começa com "weil" (porque), que introduz uma frase subordinada (veremos em níveis mais avançados).

Resposta simples no A1:
```
Warum lernst du Deutsch?
→ Ich wohne in Deutschland. (Moro na Alemanha.)
```

## 8. WIE (Como)

### 8.1. WIE (Como - Modo)

```
Wie geht es dir? (Como estás?)
→ Es geht mir gut, danke.

Wie machst du das? (Como fazes isso?)
→ Ich mache das so.

Wie kommst du zur Arbeit? (Como vais para o trabalho?)
→ Ich fahre mit dem Bus.

Wie heißt du? (Como te chamas?)
→ Ich heiße Maria.

Wie schmeckt das? (Como é o sabor disso?)
→ Das schmeckt gut!
```

### 8.2. WIE + Adjetivo (Quão/Quanto)

```
Wie alt bist du? (Quantos anos tens?)
→ Ich bin 25 Jahre alt.

Wie groß bist du? (Qual é a tua altura?)
→ Ich bin 1,75 Meter groß.

Wie lange dauert das? (Quanto tempo dura?)
→ Das dauert 2 Stunden.

Wie spät ist es? (Que horas são?)
→ Es ist 15 Uhr.

Wie viel kostet das? (Quanto custa?)
→ Das kostet 20 Euro.
```

### 8.3. WIE VIEL vs. WIE VIELE

**WIE VIEL** (quanto - singular, incontável):
```
Wie viel Geld hast du? (Quanto dinheiro tens?)
Wie viel Zeit haben wir? (Quanto tempo temos?)
Wie viel kostet das? (Quanto custa?)
Wie viel Zucker möchtest du? (Quanto açúcar queres?)
```

**WIE VIELE** (quantos - plural, contável):
```
Wie viele Kinder hast du? (Quantos filhos tens?)
Wie viele Bücher liest du? (Quantos livros lês?)
Wie viele Stunden arbeitest du? (Quantas horas trabalhas?)
Wie viele Sprachen sprichst du? (Quantas línguas falas?)
```

## 9. WELCHER, WELCHE, WELCHES (Qual)

### 9.1. Declinação

"Welcher" declina-se como artigo definido:

| Caso | Masculino | Feminino | Neutro | Plural |
|------|-----------|----------|--------|--------|
| **Nominativo** | welcher | welche | welches | welche |
| **Acusativo** | welchen | welche | welches | welche |

### 9.2. Uso

Pergunta sobre **escolha específica** entre várias opções:

```
Welcher Mann kommt? (Qual homem vem?)
→ Der große Mann kommt.

Welche Frau ist das? (Qual mulher é essa?)
→ Das ist meine Lehrerin.

Welches Buch liest du? (Qual livro lês?)
→ Ich lese das rote Buch.

Welche Bücher magst du? (Quais livros gostas?)
→ Ich mag Krimis.

Welchen Film sehen wir? (Qual filme vemos?)
→ Wir sehen den neuen Film. (acusativo masculino!)
```

## 10. Diálogos Práticos

### Diálogo 1: Conhecendo Alguém
```
A: Hallo! Wie heißt du?
   (Olá! Como te chamas?)

B: Ich heiße Maria. Und du?
   (Chamo-me Maria. E tu?)

A: Ich heiße Peter. Woher kommst du?
   (Chamo-me Peter. De onde és?)

B: Ich komme aus Portugal. Und du? Woher bist du?
   (Sou de Portugal. E tu? De onde és?)

A: Ich bin aus Deutschland. Was machst du hier?
   (Sou da Alemanha. O que fazes aqui?)

B: Ich lerne Deutsch. Wie lange wohnst du schon hier?
   (Aprendo alemão. Há quanto tempo moras aqui?)

A: Ich wohne seit 5 Jahren hier.
   (Moro aqui há 5 anos.)
```

### Diálogo 2: Planeando Encontro
```
A: Gehst du heute ins Kino?
   (Vais hoje ao cinema?)

B: Ja, ich gehe ins Kino. Kommst du mit?
   (Sim, vou ao cinema. Vens?)

A: Ja, gerne! Wann beginnt der Film?
   (Sim, com prazer! Quando começa o filme?)

B: Der Film beginnt um 20 Uhr. Wo treffen wir uns?
   (O filme começa às 20 horas. Onde nos encontramos?)

A: Treffen wir uns vor dem Kino?
   (Encontramo-nos em frente ao cinema?)

B: Perfekt! Um wie viel Uhr?
   (Perfeito! A que horas?)

A: Um 19:30 Uhr. Passt das?
   (Às 19h30. Serve?)

B: Ja, das passt. Bis später!
   (Sim, serve. Até logo!)
```

### Diálogo 3: No Supermercado
```
A: Entschuldigung, wo finde ich Milch?
   (Desculpe, onde encontro leite?)

B: Die Milch ist in Gang 3.
   (O leite está no corredor 3.)

A: Danke! Und wie viel kostet die Milch?
   (Obrigado! E quanto custa o leite?)

B: Das kommt darauf an. Welche Milch möchten Sie?
   (Depende. Que leite deseja?)

A: Ich möchte Vollmilch.
   (Quero leite gordo.)

B: Die Vollmilch kostet 1,20 Euro.
   (O leite gordo custa 1,20 euros.)
```

## 13. Erros Comuns e Como Evitá-los

### Erro 1: Ordem errada em perguntas sim/não
**Incorreto:** *Du sprichst Deutsch?* (com entoação de afirmação)
**Correto:** Sprichst du Deutsch?

**Solução:** O verbo deve ir para a primeira posição.

### Erro 2: Confundir "wo", "wohin" e "woher"
**Incorreto:** *Wo gehst du?* (quando pergunta direção)
**Correto:** Wohin gehst du?

**Solução:** Wo = onde está, Wohin = para onde vai, Woher = de onde vem.

### Erro 3: Usar "was" em vez de "wer" para pessoas
**Incorreto:** *Was ist das?* (referindo-se a uma pessoa)
**Correto:** Wer ist das?

**Solução:** "Was" para coisas, "wer" para pessoas.

### Erro 4: Confundir "wie viel" e "wie viele"
**Incorreto:** *Wie viele Geld hast du?*
**Correto:** Wie viel Geld hast du?

**Solução:** "Wie viel" para incontáveis, "wie viele" para contáveis.

### Erro 5: Não separar o prefixo em verbos separáveis
**Incorreto:** *Wann ankommst du?*
**Correto:** Wann kommst du an?

**Solução:** O prefixo vai para o final.

## 14. Resumo da Lição

### Pontos-Chave:

1. **Perguntas Sim/Não:** Verbo em 1ª posição
   - Sprichst du Deutsch?

2. **Perguntas W-:** Pronome interrogativo + verbo + sujeito
   - Was machst du?

3. **Pronomes principais:**
   - wer/wen (quem), was (o quê)
   - wo/wohin/woher (onde)
   - wann (quando), warum (por quê)
   - wie (como), wie viel/wie viele (quanto/quantos)
   - welcher (qual)

4. **Resposta "doch":** Para contradizer perguntas negativas

5. **Ordem com verbos separáveis:** Prefixo no final

## 15. Linguagem Formal, Informal e de Rua

As perguntas são essenciais na comunicação quotidiana. Vamos ver como os alemães fazem perguntas em diferentes contextos.

### Situação: Perguntar o Nome

#### **Formal**
```
Wie heißen Sie?
Wie ist Ihr Name?
Darf ich Ihren Namen erfahren?
```
**Tradução:** Como se chama? / Qual é o seu nome? / Posso saber o seu nome?

#### **Informal**
```
Wie heißt du?
Wie ist dein Name?
```
**Tradução:** Como te chamas? / Qual é o teu nome?

#### **Rua/Jovem**
```
Wie heißt du?
Wie heißte nochmal? (quando esqueceste)
Und du so? (E tu?)
```
**Tradução:** Como te chamas? / Como é que te chamas mesmo? / E tu?

**Gírias:**
- **Und du so?** = E tu? (muito coloquial, literalmente "E tu assim?")
- **nochmal** = de novo, outra vez (usado quando esqueces algo)

### Situação: Perguntar Onde Mora

#### **Formal**
```
Wo wohnen Sie?
In welcher Stadt leben Sie?
```
**Tradução:** Onde mora? / Em que cidade vive?

#### **Informal**
```
Wo wohnst du?
Wo lebst du?
```
**Tradução:** Onde moras? / Onde vives?

#### **Rua/Jovem**
```
Wo wohnste?
Wo kommste her?
Wo biste zu Hause?
```
**Tradução:** Onde moras? / De onde és? / Onde é a tua casa?

**Contrações coloquiais:**
- **wohnste** = "wohnst du" (moras tu)
- **kommste** = "kommst du" (vens tu)
- **biste** = "bist du" (és tu)

### Situação: Perguntar o Que Alguém Está Fazendo

#### **Formal**
```
Was machen Sie gerade?
Womit beschäftigen Sie sich?
```
**Tradução:** O que está a fazer agora? / Com o que se ocupa?

#### **Informal**
```
Was machst du gerade?
Was machst du so?
```
**Tradução:** O que estás a fazer agora? / O que andas a fazer?

#### **Rua/Jovem**
```
Was machste?
Was geht ab?
Was läuft?
Was geht bei dir?
```
**Tradução:** O que fazes? / O que se passa? / Como vão as coisas? / Como vão as coisas contigo?

**Expressões de rua:**
- **Was geht ab?** = O que se passa? / E aí?
- **Was läuft?** = Como vão as coisas? (literalmente "O que corre?")
- **Was geht bei dir?** = Como vão as coisas contigo?

### Situação: Perguntar Se Alguém Quer Algo

#### **Formal**
```
Möchten Sie etwas trinken?
Hätten Sie gerne einen Kaffee?
Darf ich Ihnen etwas anbieten?
```
**Tradução:** Quer beber algo? / Gostaria de um café? / Posso oferecer-lhe algo?

#### **Informal**
```
Möchtest du etwas trinken?
Willst du einen Kaffee?
Magst du Kaffee?
```
**Tradução:** Queres beber algo? / Queres um café? / Gostas de café?

#### **Rua/Jovem**
```
Willste was trinken?
Haste Bock auf 'n Kaffee?
Lust auf 'n Bier?
```
**Tradução:** Queres beber algo? / Tens vontade de um café? / Vontade de uma cerveja?

**Expressões coloquiais:**
- **Willste** = "Willst du" (queres tu)
- **Haste** = "Hast du" (tens tu)
- **Bock auf** = vontade de (gíria)
- **'n** = "ein" (um)
- **Lust auf** = vontade de

### Situação: Perguntar as Horas

#### **Formal**
```
Wie spät ist es?
Können Sie mir sagen, wie spät es ist?
Wissen Sie, wie viel Uhr es ist?
```
**Tradução:** Que horas são? / Pode dizer-me que horas são? / Sabe que horas são?

#### **Informal**
```
Wie spät ist es?
Wie viel Uhr ist es?
Hast du eine Uhr?
```
**Tradução:** Que horas são? / Que horas são? / Tens relógio?

#### **Rua/Jovem**
```
Wie spät isse?
Haste mal die Uhrzeit?
Weißte wie spät es is?
```
**Tradução:** Que horas são? / Tens as horas? / Sabes que horas são?

**Contrações extremas:**
- **isse** = "ist es" (é)
- **Haste mal** = "Hast du mal" (tens)
- **Weißte** = "Weißt du" (sabes tu)
- **is** = "ist" (é)

### Situação: Perguntar Preços

#### **Formal**
```
Was kostet das?
Wie viel kostet das, bitte?
Können Sie mir den Preis sagen?
```
**Tradução:** Quanto custa isto? / Quanto custa isto, por favor? / Pode dizer-me o preço?

#### **Informal**
```
Was kostet das?
Wie viel kostet das?
Wie teuer ist das?
```
**Tradução:** Quanto custa isto? / Quanto custa isto? / Quão caro é isto?

#### **Rua/Jovem**
```
Was kostet der Scheiß? (rude)
Wie viel willste dafür?
Was soll das kosten?
```
**Tradução:** Quanto custa esta merda? (rude) / Quanto queres por isso? / Quanto é que isso há de custar?

**Nota:** "der Scheiß" é muito rude, evitar em contextos educados.

### Situação: Perguntar Se Alguém Entende

#### **Formal**
```
Verstehen Sie das?
Ist das verständlich?
Haben Sie Fragen?
```
**Tradução:** Compreende isto? / Isto é compreensível? / Tem perguntas?

#### **Informal**
```
Verstehst du das?
Kapierst du das?
Alles klar?
```
**Tradução:** Compreendes isto? / Percebes isto? / Tudo claro?

#### **Rua/Jovem**
```
Checkst du das?
Peilst du das?
Kapiert?
Check?
```
**Tradução:** Sacas isto? / Percebes isto? / Percebido? / Sacou?

**Gírias para "entender":**
- **checken** = sacar, entender (do inglês "check")
- **kapieren** = perceber (coloquial)
- **peilen** = captar, perceber (muito coloquial)
- **raffen** = sacar, entender (gíria)

### Tabela Comparativa de Perguntas Comuns

| Formal | Informal | Rua/Jovem | Tradução |
|--------|----------|-----------|----------|
| Wie heißen Sie? | Wie heißt du? | Und du so? | Como se chama? |
| Was machen Sie? | Was machst du? | Was geht ab? | O que faz? / E aí? |
| Möchten Sie...? | Willst du...? | Haste Bock auf...? | Quer...? / Tens vontade de...? |
| Verstehen Sie? | Verstehst du? | Checkst du das? | Compreende? / Sacas? |
| Wie spät ist es? | Wie spät ist es? | Wie spät isse? | Que horas são? |
| Wo wohnen Sie? | Wo wohnst du? | Wo wohnste? | Onde mora? |

**Nota Cultural:** Em contextos jovens/urbanos, as contrações são extremamente comuns. "Hast du" vira "Haste", "Bist du" vira "Biste", etc. Também é comum eliminar o "t" final: "ist" vira "is".
