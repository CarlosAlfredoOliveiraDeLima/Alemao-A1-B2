# Preposições de Acusativo (durch, für, gegen, ohne, um)

## Introdução

As preposições em alemão são palavras que estabelecem relações entre elementos da frase, indicando direção, lugar, tempo, modo, etc. Uma característica fundamental das preposições alemãs é que cada uma rege um caso específico: algumas regem acusativo, outras dativo, e algumas podem reger ambos dependendo do contexto.

Nesta lição, vamos focar nas cinco preposições que **sempre** regem o caso acusativo: durch, für, gegen, ohne e um. Estas cinco são fáceis de memorizar com a sigla mnemónica **"DOGFU"** (durch, ohne, gegen, für, um) ou **"FUDOG"** (für, um, durch, ohne, gegen).

---

## 1. Visão Geral das Preposições de Acusativo

### 1.1. As Cinco Preposições

| Preposição | Significado Principal | Uso Comum |
|------------|----------------------|-----------|
| **durch** | através de, por | movimento através de algo |
| **für** | para, por | destinatário, propósito, duração |
| **gegen** | contra | oposição, direção, tempo aproximado |
| **ohne** | sem | ausência |
| **um** | à volta de, às (tempo) | movimento circular, hora exata |

### 1.2. Regra Fundamental

**Todas estas preposições exigem SEMPRE o caso acusativo!**

Isto significa que após estas preposições:
- Masculino singular: der → **den**, ein → **einen**
- Feminino, neutro e plural: permanecem iguais ao nominativo

### 1.3. Mnemónica DOGFU/FUDOG

Para memorizar as cinco preposições:
```
D - durch
O - ohne
G - gegen
F - für
U - um
```

Ou reordenando: **F**ür, **U**m, **D**urch, **O**hne, **G**egen = **FUDOG**

## 2. DURCH (através de, por)

### 2.1. Significados e Usos

**A. Movimento através de um espaço:**
```
durch die Stadt — através da cidade
durch den Wald — através da floresta
durch das Fenster — pela janela
durch die Tür — pela porta
```

**B. Meio ou agente (passiva):**
```
durch den Lehrer — pelo professor (ensinado pelo professor)
durch das Internet — pela internet/através da internet
durch einen Brief — por carta
```

**C. Causa:**
```
durch den Regen — por causa da chuva
durch einen Fehler — devido a um erro
```

### 2.2. Exemplos Detalhados

#### Com Artigos Definidos
```
Wir gehen durch den Park. (Vamos através do parque.)
— der Park → den Park (masculino, acusativo)

Sie fährt durch die Stadt. (Ela conduz através da cidade.)
— die Stadt → die Stadt (feminino, não muda)

Das Licht kommt durch das Fenster. (A luz vem pela janela.)
— das Fenster → das Fenster (neutro, não muda)

Wir laufen durch die Straßen. (Corremos pelas ruas.)
— die Straßen → die Straßen (plural, não muda)
```

#### Com Artigos Indefinidos
```
Wir gehen durch einen Tunnel. (Vamos através de um túnel.)
— ein Tunnel → einen Tunnel (masculino!)

Er schaut durch eine Tür. (Ele espreita por uma porta.)
— eine Tür → eine Tür (feminino, não muda)

Sie geht durch ein Tor. (Ela passa por um portão.)
— ein Tor → ein Tor (neutro, não muda)
```

#### Em Contexto
```
Ich lerne Deutsch durch Übung. (Aprendo alemão através da prática.)
Der Brief kommt durch die Post. (A carta vem pelo correio.)
Wir fahren durch den Wald. (Conduzimos através da floresta.)
Er blickt durch das Fenster. (Ele olha pela janela.)
Durch den Regen bin ich nass. (Por causa da chuva estou molhado.)
```

## 3. FÜR (para, por)

### 3.1. Significados e Usos

**A. Destinatário ou beneficiário:**
```
für meinen Bruder — para o meu irmão
für die Kinder — para as crianças
für dich — para ti
```

**B. Propósito ou objetivo:**
```
für die Arbeit — para o trabalho
für die Prüfung — para o exame
```

**C. Duração de tempo (futuro):**
```
für einen Monat — por um mês
für zwei Wochen — por duas semanas
für ein Jahr — por um ano
```

**D. A favor de:**
```
für die Idee — a favor da ideia
für den Plan — a favor do plano
```

**E. Em vez de:**
```
für mich — por mim (em vez de mim)
```

**F. Preço:**
```
für 10 Euro — por 10 euros
```

### 3.2. Exemplos Detalhados

#### Destinatário
```
Das Geschenk ist für meinen Vater. (O presente é para o meu pai.)
— mein Vater → meinen Vater (masculino!)

Ich kaufe Blumen für meine Mutter. (Compro flores para a minha mãe.)
— meine Mutter → meine Mutter (feminino, não muda)

Das ist für das Kind. (Isto é para a criança.)
— das Kind → das Kind (neutro, não muda)

Die Bücher sind für die Studenten. (Os livros são para os estudantes.)
— die Studenten → die Studenten (plural, não muda)
```

#### Propósito
```
Ich lerne für die Prüfung. (Estudo para o exame.)
Er arbeitet für das Projekt. (Ele trabalha para o projeto.)
Wir sparen für den Urlaub. (Poupamos para as férias.)
Sie trainiert für den Marathon. (Ela treina para a maratona.)
```

#### Duração
```
Ich fahre für einen Monat nach Deutschland. (Vou para a Alemanha por um mês.)
Er bleibt für eine Woche. (Ele fica por uma semana.)
Wir mieten das Haus für ein Jahr. (Alugamos a casa por um ano.)
```

#### Preço
```
Ich kaufe das Buch für 20 Euro. (Compro o livro por 20 euros.)
Das Auto kostet für einen guten Preis. (O carro custa por um bom preço.)
```

#### Expressões Idiomáticas
```
Schritt für Schritt — passo a passo
Tag für Tag — dia após dia
für immer — para sempre
für mich — para mim / na minha opinião
```

## 4. GEGEN (contra)

### 4.1. Significados e Usos

**A. Oposição física:**
```
gegen die Wand — contra a parede
gegen den Baum — contra a árvore
```

**B. Oposição abstrata (ser contra):**
```
gegen die Idee — contra a ideia
gegen den Plan — contra o plano
```

**C. Direção de movimento (para/em direção a):**
```
gegen Norden — para o norte
```

**D. Tempo aproximado:**
```
gegen 5 Uhr — pelas 5 horas / por volta das 5
gegen Abend — ao anoitecer
gegen Mittag — por volta do meio-dia
```

**E. Comparação (contra, em comparação com):**
```
gegen den anderen — comparado com o outro
```

**F. Troca:**
```
gegen Geld — em troca de dinheiro
```

**G. Remédio:**
```
gegen Kopfschmerzen — para dores de cabeça
gegen Erkältung — para constipações
```

### 4.2. Exemplos Detalhados

#### Oposição Física
```
Er läuft gegen den Baum. (Ele corre contra a árvore.)
— der Baum → den Baum (masculino!)

Das Auto fährt gegen die Wand. (O carro embate contra a parede.)
— die Wand → die Wand (feminino, não muda)

Der Ball fliegt gegen das Fenster. (A bola voa contra a janela.)
— das Fenster → das Fenster (neutro, não muda)
```

#### Oposição Abstrata
```
Ich bin gegen den Plan. (Sou contra o plano.)
Sie ist gegen die Idee. (Ela é contra a ideia.)
Wir kämpfen gegen das Problem. (Lutamos contra o problema.)
Er protestiert gegen die Regierung. (Ele protesta contra o governo.)
```

#### Tempo Aproximado
```
Ich komme gegen 8 Uhr. (Venho por volta das 8 horas.)
Wir treffen uns gegen Mittag. (Encontramo-nos por volta do meio-dia.)
Er schläft gegen Abend. (Ele dorme ao anoitecer.)
Sie kommt gegen Ende des Monats. (Ela vem para o fim do mês.)
```

#### Remédio
```
Das ist ein Medikament gegen Kopfschmerzen. (Isto é um medicamento para dores de cabeça.)
— die Kopfschmerzen → die Kopfschmerzen (plural)

Ich nehme Tabletten gegen die Erkältung. (Tomo comprimidos para a constipação.)
— die Erkältung → die Erkältung (feminino)
```

#### Expressões Comuns
```
gegen den Strom schwimmen — nadar contra a corrente
gegen die Regeln — contra as regras
Tor gegen Tor — golo contra golo (empate)
```

## 5. OHNE (sem)

### 5.1. Significados e Usos

**A. Ausência de algo ou alguém:**
```
ohne meinen Bruder — sem o meu irmão
ohne Geld — sem dinheiro
ohne Hilfe — sem ajuda
```

**B. Sem fazer algo (+ infinitivo em alemão avançado):**
```
ohne zu fragen — sem perguntar
```

### 5.2. Exemplos Detalhados

#### Ausência de Pessoa
```
Ich gehe ohne meinen Freund. (Vou sem o meu amigo.)
— mein Freund → meinen Freund (masculino!)

Er kommt ohne seine Frau. (Ele vem sem a mulher dele.)
— seine Frau → seine Frau (feminino, não muda)

Wir fahren ohne das Kind. (Vamos sem a criança.)
— das Kind → das Kind (neutro, não muda)

Sie reisen ohne die Eltern. (Eles viajam sem os pais.)
— die Eltern → die Eltern (plural, não muda)
```

#### Ausência de Objeto/Conceito
```
Ich trinke Kaffee ohne Zucker. (Bebo café sem açúcar.)
Er arbeitet ohne Pause. (Ele trabalha sem pausa.)
Sie lebt ohne Geld. (Ela vive sem dinheiro.)
Wir lernen ohne Buch. (Aprendemos sem livro.)
```

**Nota:** Quando não há artigo na afirmativa, também não há no acusativo:
```
ohne Geld (sem dinheiro) — não "ohne das Geld"
ohne Hilfe (sem ajuda) — não "ohne die Hilfe"
```

**Mas:**
```
ohne den Computer (sem o computador) — com artigo definido
ohne einen Computer (sem um computador) — com artigo indefinido
```

#### Expressões Comuns
```
ohne Zweifel — sem dúvida
ohne Frage — sem questão / sem dúvida
ohne Ende — sem fim
ohne Probleme — sem problemas
ohne weiteres — sem mais nem menos
```

## 6. UM (à volta de, às)

### 6.1. Significados e Usos

**A. Hora exata:**
```
um 8 Uhr — às 8 horas
um Mitternacht — à meia-noite
```

**B. Movimento circular (à volta de):**
```
um das Haus — à volta da casa
um die Ecke — à volta da esquina
```

**C. Aproximadamente (com números):**
```
um 50 Euro — cerca de 50 euros
um 100 Leute — cerca de 100 pessoas
```

**D. Sobre, acerca de (pedir, lutar por):**
```
um Hilfe bitten — pedir ajuda
um den Job kämpfen — lutar pelo emprego
```

**E. Diferença (em expressões de comparação):**
```
um 5 Euro teurer — 5 euros mais caro
```

### 6.2. Exemplos Detalhados

#### Hora Exata
```
Der Unterricht beginnt um 9 Uhr. (A aula começa às 9 horas.)
Ich komme um 8 Uhr. (Venho às 8 horas.)
Wir treffen uns um Mittag. (Encontramo-nos ao meio-dia.)
Sie schläft um Mitternacht. (Ela dorme à meia-noite.)
Der Film startet um 20 Uhr. (O filme começa às 20 horas.)
```

#### Movimento Circular
```
Wir gehen um den See. (Andamos à volta do lago.)
— der See → den See (masculino!)

Er läuft um die Ecke. (Ele corre à volta da esquina.)
— die Ecke → die Ecke (feminino, não muda)

Das Auto fährt um das Haus. (O carro conduz à volta da casa.)
— das Haus → das Haus (neutro, não muda)

Sie tanzen um den Baum. (Eles dançam à volta da árvore.)
— der Baum → den Baum (masculino!)
```

#### Aproximadamente
```
Das kostet um 50 Euro. (Isso custa cerca de 50 euros.)
Es sind um 100 Leute hier. (Há cerca de 100 pessoas aqui.)
Ich brauche um 2 Stunden. (Preciso de cerca de 2 horas.)
```

#### Pedir, Lutar Por
```
Ich bitte um Hilfe. (Peço ajuda.)
Er kämpft um den Job. (Ele luta pelo emprego.)
Sie bittet um Verständnis. (Ela pede compreensão.)
Wir spielen um den Titel. (Jogamos pelo título.)
```

#### Expressões Comuns
```
um die Ecke — à volta da esquina
um diese Zeit — por esta altura
Schritt um Schritt — passo a passo
um Gottes willen — pelo amor de Deus
um jeden Preis — a qualquer preço
```

## 7. Contrações Comuns

### 7.1. Preposição + Artigo

Algumas preposições de acusativo contraem-se com o artigo "das":

| Preposição + Artigo | Contração | Uso |
|---------------------|-----------|-----|
| durch + das | durchs | durchs Zimmer (pelo quarto) |
| für + das | fürs | fürs Kind (para a criança) |
| um + das | ums | ums Haus (à volta da casa) |

**Exemplos:**
```
Formal: Wir gehen durch das Zimmer.
Coloquial: Wir gehen durchs Zimmer.
(Vamos pelo quarto.)

Formal: Das ist für das Baby.
Coloquial: Das ist fürs Baby.
(Isto é para o bebé.)

Formal: Er läuft um das Gebäude.
Coloquial: Er läuft ums Gebäude.
(Ele corre à volta do edifício.)
```

**Nota:** Estas contrações são opcionais e mais comuns na fala coloquial.

## 9. Diálogos Práticos

### Diálogo 1: Marcando um Encontro
```
A: Wann treffen wir uns?
   (Quando nos encontramos?)

B: Um 7 Uhr am Bahnhof?
   (Às 7 no terminal?)

A: Okay. Gehst du auch durch den Park?
   (Está bem. Também vais pelo parque?)

B: Nein, ich gehe um den Park. Das ist schneller.
   (Não, vou à volta do parque. É mais rápido.)
```

### Diálogo 2: Compras
```
A: Für wen kaufst du das Geschenk?
   (Para quem compras o presente?)

B: Für meinen Bruder. Er hat morgen Geburtstag.
   (Para o meu irmão. Ele faz anos amanhã.)

A: Was kostet es?
   (Quanto custa?)

B: Um 50 Euro.
   (Cerca de 50 euros.)
```

### Diálogo 3: No Médico
```
A: Was haben Sie gegen Kopfschmerzen?
   (O que tem para dores de cabeça?)

B: Diese Tabletten sind sehr gut. Nehmen Sie eine ohne Wasser.
   (Estes comprimidos são muito bons. Tome um sem água.)

A: Und wie oft?
   (E com que frequência?)

B: Alle 6 Stunden, also um 8, um 14, um 20 Uhr...
   (De 6 em 6 horas, portanto às 8, às 14, às 20...)
```

## 12. Erros Comuns e Como Evitá-los

### Erro 1: Usar nominativo em vez de acusativo
**Incorreto:** *durch der Park*
**Correto:** durch den Park

**Solução:** Lembre-se: estas cinco preposições SEMPRE regem acusativo.

### Erro 2: Esquecer de mudar o masculino
**Incorreto:** *für mein Vater*
**Correto:** für meinen Vater

**Solução:** Masculino muda no acusativo (der→den, mein→meinen).

### Erro 3: Confundir "um" (hora) com "gegen" (aproximado)
**Incorreto:** *Ich komme gegen 8 Uhr* (quando quer dizer exatamente às 8)
**Correto:** Ich komme um 8 Uhr. (hora exata)

**Nota:** "gegen 8 Uhr" significa "por volta das 8", não exatamente às 8.

### Erro 4: Usar artigo desnecessário com "ohne"
**Menos natural:** *ohne das Geld*
**Mais natural:** ohne Geld (quando é conceito geral)

**Mas:** ohne das Geld (quando se refere a um dinheiro específico)

## 13. Resumo da Lição

### Pontos-Chave:

1. **Cinco preposições de acusativo:** durch, für, gegen, ohne, um (DOGFU/FUDOG)

2. **Sempre acusativo:** estas preposições NUNCA regem outro caso

3. **Masculino muda:** der→den, ein→einen, mein→meinen

4. **Significados principais:**
   - durch = através de, por
   - für = para, por (duração)
   - gegen = contra, por volta de (tempo)
   - ohne = sem
   - um = às (hora exata), à volta de

5. **Contrações comuns:** durchs, fürs, ums (coloquiais)
