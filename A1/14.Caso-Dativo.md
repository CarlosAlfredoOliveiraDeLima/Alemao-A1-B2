# Caso Dativo - Introdução

## Introdução

O caso Dativo (Dativ) é o terceiro caso gramatical do alemão e um dos mais importantes para a comunicação quotidiana. Enquanto o Acusativo indica o objeto direto (o quê/quem recebe a ação diretamente), o Dativo indica o objeto indireto (a quem/para quem se faz algo) e é usado com muitas preposições comuns.

O Dativo responde às perguntas "**wem?**" (a quem?) e "**wo?**" (onde? - com preposições de localização). Nesta lição, vamos introduzir o Dativo e aprender como usá-lo corretamente.

---

## 1. O que é o Caso Dativo?

### 1.1. Função Principal

O Dativo indica:
1. **Objeto indireto** (a quem, para quem)
2. **Localização** (com certas preposições)
3. **Beneficiário** de uma ação
4. **Posse** (em construções especiais)

### 1.2. Pergunta-Chave

**Wem?** (A quem?)

```
Ich gebe dem Mann das Buch.
(Dou o livro ao homem.)

Wem gibst du das Buch? (A quem dás o livro?)
→ Dem Mann. (Ao homem.)
```

## 2. Artigos no Dativo

### 2.1. Artigos Definidos

| Género | Nominativo | Acusativo | **Dativo** |
|--------|------------|-----------|------------|
| **Masculino** | der | den | **dem** |
| **Feminino** | die | die | **der** |
| **Neutro** | das | das | **dem** |
| **Plural** | die | die | **den** (+ n) |

**Mudanças cruciais:**
- Masculino e neutro: → **dem**
- Feminino: → **der**
- Plural: → **den** (e o substantivo ganha **-n** se ainda não tiver)

### 2.2. Artigos Indefinidos e Negativos

| Género | Nominativo | Acusativo | **Dativo** |
|--------|------------|-----------|------------|
| **Masculino** | ein/kein | einen/keinen | **einem/keinem** |
| **Feminino** | eine/keine | eine/keine | **einer/keiner** |
| **Neutro** | ein/kein | ein/kein | **einem/keinem** |
| **Plural** | —/keine | —/keine | **—/keinen** |

### 2.3. Exemplos Comparativos

```
Nominativo: Der Mann ist hier. (O homem está aqui.)
Acusativo: Ich sehe den Mann. (Vejo o homem.)
Dativo: Ich gebe dem Mann ein Buch. (Dou um livro ao homem.)

Nominativo: Die Frau kommt. (A mulher vem.)
Acusativo: Ich kenne die Frau. (Conheço a mulher.)
Dativo: Ich helfe der Frau. (Ajudo a mulher.)

Nominativo: Das Kind spielt. (A criança brinca.)
Acusativo: Ich sehe das Kind. (Vejo a criança.)
Dativo: Ich gebe dem Kind ein Spielzeug. (Dou um brinquedo à criança.)
```

---

## 3. Substantivos no Dativo Plural

### 3.1. Regra Especial

**No dativo plural, os substantivos ganham -n no final (se ainda não tiverem):**

```
Nominativo: die Kinder (as crianças)
Dativo: den Kindern (às crianças)

Nominativo: die Männer (os homens)
Dativo: den Männern (aos homens)

Nominativo: die Frauen (as mulheres)
Dativo: den Frauen (às mulheres) — já termina em -n, não muda!
```

### 3.2. Exemplos

```
Ich helfe den Kindern. (Ajudo as crianças.)
Ich gebe den Männern das Buch. (Dou o livro aos homens.)
Er dankt den Frauen. (Ele agradece às mulheres.)
Wir schreiben den Lehrern. (Escrevemos aos professores.)
```

**Exceção:** Substantivos que já terminam em -n ou -s não mudam:
```
die Frauen → den Frauen (já tem -n)
die Autos → den Autos (termina em -s)
```

## 4. Pronomes Pessoais no Dativo

### 4.1. Tabela Completa

| Nominativo | Acusativo | **Dativo** | Português |
|------------|-----------|------------|-----------|
| **ich** | mich | **mir** | (a) mim/me |
| **du** | dich | **dir** | (a) ti/te |
| **er** | ihn | **ihm** | (a) ele/lhe |
| **sie** | sie | **ihr** | (a) ela/lhe |
| **es** | es | **ihm** | (a) ele/lhe (neutro) |
| **wir** | uns | **uns** | (a) nós/nos |
| **ihr** | euch | **euch** | (a) vós/vos |
| **sie** | sie | **ihnen** | (a) eles/lhes |
| **Sie** | Sie | **Ihnen** | (a) você/lhe |

**Observações:**
- "wir" e "ihr": mesma forma no acusativo e dativo (uns, euch)
- Todos os outros mudam

### 4.2. Exemplos com Pronomes

```
Er hilft mir. (Ele ajuda-me.)
Ich gebe dir das Buch. (Dou-te o livro.)
Wir danken ihm. (Agradecemos-lhe.)
Sie schreibt ihr einen Brief. (Ela escreve-lhe uma carta.)
Gibst du uns das Geld? (Dás-nos o dinheiro?)
Ich zeige euch den Weg. (Mostro-vos o caminho.)
Er erklärt ihnen das Problem. (Ele explica-lhes o problema.)
Können Sie mir helfen? (Pode ajudar-me?)
```

## 6. Verbos que Regem Dativo

### 6.1. Verbos Comuns com Dativo

Muitos verbos alemães exigem o caso dativo para o seu complemento. Estes devem ser memorizados:

| Verbo | Significado | Exemplo |
|-------|-------------|---------|
| **helfen** | ajudar | Ich helfe dem Mann. |
| **danken** | agradecer | Er dankt der Frau. |
| **geben** | dar | Ich gebe ihm das Buch. |
| **zeigen** | mostrar | Sie zeigt mir den Weg. |
| **schreiben** | escrever (a alguém) | Wir schreiben den Eltern. |
| **gehören** | pertencer | Das Buch gehört mir. |
| **gefallen** | agradar | Der Film gefällt mir. |
| **schmecken** | saber bem | Das Essen schmeckt ihm. |
| **antworten** | responder | Sie antwortet dem Lehrer. |
| **folgen** | seguir | Ich folge dir. |
| **gratulieren** | felicitar | Wir gratulieren ihm. |
| **passen** | servir/caber | Die Hose passt mir nicht. |
| **fehlen** | faltar | Du fehlst mir. |
| **glauben** | acreditar | Ich glaube dir. |
| **vertrauen** | confiar | Er vertraut mir. |

### 6.2. Exemplos Detalhados

#### HELFEN (ajudar)
```
Ich helfe dem Lehrer. (Ajudo o professor.)
Kannst du mir helfen? (Podes ajudar-me?)
Wir helfen den Kindern. (Ajudamos as crianças.)
Er hilft seiner Mutter. (Ele ajuda a mãe dele.)
```

#### DANKEN (agradecer)
```
Ich danke dir. (Agradeço-te.)
Er dankt dem Arzt. (Ele agradece ao médico.)
Wir danken Ihnen. (Agradecemos-lhe.) — formal
Sie dankt ihren Freunden. (Ela agradece aos amigos dela.)
```

#### GEFALLEN (agradar)
```
Der Film gefällt mir. (O filme agrada-me / Gosto do filme.)
Gefällt dir das Buch? (Gostas do livro?)
Die Stadt gefällt ihm. (Ele gosta da cidade.)
Die Bücher gefallen uns. (Gostamos dos livros.)
```

**Nota:** "gefallen" tem estrutura diferente do português:
- Alemão: "O filme me agrada" (o filme é o sujeito)
- Português: "Eu gosto do filme" (eu sou o sujeito)

#### GEHÖREN (pertencer)
```
Das Buch gehört mir. (O livro pertence-me.)
Gehört dir das Auto? (O carro pertence-te?)
Die Tasche gehört ihr. (A mala pertence-lhe.)
Das Haus gehört meinem Vater. (A casa pertence ao meu pai.)
```

#### SCHREIBEN (escrever a alguém)
```
Ich schreibe dir eine E-Mail. (Escrevo-te um email.)
Er schreibt seinem Freund einen Brief. (Ele escreve uma carta ao amigo dele.)
Wir schreiben den Eltern. (Escrevemos aos pais.)
```

## 7. Construções com Dois Objetos

### 7.1. Ordem: Dativo + Acusativo

Muitos verbos têm **dois objetos**: um indireto (dativo - a quem) e um direto (acusativo - o quê):

**Estrutura com substantivos:**
```
Sujeito + Verbo + Dativo (a quem) + Acusativo (o quê)

Ich gebe dem Mann das Buch.
(Dou ao homem o livro.)

Er zeigt der Frau den Weg.
(Ele mostra à mulher o caminho.)

Wir kaufen den Kindern Spielzeug.
(Compramos brinquedos às crianças.)
```

### 7.2. Ordem: Pronome Acusativo + Dativo

**Quando o acusativo é pronome, vem primeiro:**

```
Ich gebe es ihm. (Dou-o a ele.)
— não: Ich gebe ihm es.

Er zeigt sie mir. (Ele mostra-ma.)
Sie kauft ihn uns. (Ela compra-no para nós.)
```

### 7.3. Exemplos Comparativos

| Dois Substantivos | Pronome Acusativo |
|-------------------|-------------------|
| Ich gebe dem Mann das Buch. | Ich gebe **es** **ihm**. |
| Er zeigt der Frau den Weg. | Er zeigt **ihn** **ihr**. |
| Wir schreiben den Eltern einen Brief. | Wir schreiben **ihn** **ihnen**. |

## 8. Preposições com Dativo (Introdução)

### 8.1. Preposições que Sempre Regem Dativo

Existem preposições que **sempre** usam dativo (veremos em detalhe na lição 15):

```
aus — de (origem), feito de
bei — em casa de, junto de
mit — com
nach — depois de, para (países/cidades)
seit — desde
von — de (posse, origem)
zu — para, a
```

### 8.2. Exemplos Rápidos

```
Ich komme aus dem Haus. (Venho de casa.)
Er wohnt bei seinen Eltern. (Ele mora com os pais dele.)
Wir fahren mit dem Auto. (Vamos de carro.)
Sie geht nach der Schule nach Hause. (Ela vai para casa depois da escola.)
Ich lerne seit einem Monat Deutsch. (Aprendo alemão há um mês.)
Das Buch ist von meinem Bruder. (O livro é do meu irmão.)
Wir gehen zur Schule. (Vamos à escola.)
```

## 9. Diálogos Práticos

### Diálogo 1: Pedindo Ajuda
```
A: Kannst du mir helfen?
   (Podes ajudar-me?)

B: Ja, natürlich! Was kann ich für dich tun?
   (Sim, claro! O que posso fazer por ti?)

A: Ich verstehe die Hausaufgabe nicht. Kannst du sie mir erklären?
   (Não compreendo o trabalho de casa. Podes explicar-mo?)

B: Kein Problem. Ich zeige dir, wie es geht.
   (Sem problema. Mostro-te como se faz.)
```

### Diálogo 2: Agradecimentos
```
A: Hier ist dein Buch zurück. Vielen Dank!
   (Aqui está o teu livro de volta. Muito obrigado!)

B: Gern geschehen! Hat dir das Buch gefallen?
   (De nada! Gostaste do livro?)

A: Ja, es hat mir sehr gut gefallen!
   (Sim, gostei muito!)

B: Das freut mich. Ich kann dir ein anderes Buch empfehlen.
   (Fico contente. Posso recomendar-te outro livro.)
```

### Diálogo 3: No Restaurante
```
Kellner: Schmeckt Ihnen das Essen?
         (Está a gostar da comida?)

Gast: Ja, es schmeckt mir sehr gut!
      (Sim, está a saber-me muito bem!)

Kellner: Das freut mich. Kann ich Ihnen noch etwas bringen?
         (Fico contente. Posso trazer-lhe mais alguma coisa?)

Gast: Ja, bringen Sie mir bitte die Rechnung.
      (Sim, traga-me a conta, por favor.)
```

## 12. Erros Comuns e Como Evitá-los

### Erro 1: Usar acusativo em vez de dativo com verbos de dativo
**Incorreto:** *Ich helfe den Mann.*
**Correto:** Ich helfe dem Mann.

**Solução:** Memorize os verbos que regem dativo (helfen, danken, gefallen, etc.).

### Erro 2: Esquecer o -n no plural
**Incorreto:** *Ich helfe den Kinder.*
**Correto:** Ich helfe den Kindern.

**Solução:** Dativo plural sempre termina em -n (se ainda não tiver).

### Erro 3: Ordem errada com dois objetos
**Incorreto:** *Ich gebe ihm es.*
**Correto:** Ich gebe es ihm.

**Solução:** Quando acusativo é pronome, vem antes do dativo.

### Erro 4: Confundir "gefallen" com "mögen"
**Incorreto:** *Ich gefalle den Film.*
**Correto:** Der Film gefällt mir.

**Solução:** Com "gefallen", a coisa que agrada é o sujeito (nominativo), a pessoa é dativo.

## 13. Resumo da Lição

### Pontos-Chave:

1. **Dativo** = objeto indireto (a quem, para quem)
   - Pergunta: **Wem?**

2. **Artigos no dativo:**
   - Masculino/Neutro: dem/einem
   - Feminino: der/einer
   - Plural: den/keinen (+ substantivo ganha -n)

3. **Pronomes pessoais:** mir, dir, ihm, ihr, ihm, uns, euch, ihnen, Ihnen

4. **Verbos com dativo:** helfen, danken, geben, zeigen, gehören, gefallen, schmecken, etc.

5. **Dois objetos:** Dativo + Acusativo (ou pronome acusativo + dativo)

6. **Preposições de dativo:** aus, bei, mit, nach, seit, von, zu
