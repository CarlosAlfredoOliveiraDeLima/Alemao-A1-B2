# Imperativo (du, ihr, Sie)

## Introdução

O imperativo (Imperativ) é a forma verbal usada para dar ordens, instruções, conselhos ou fazer pedidos. Em alemão, existem três formas principais de imperativo, correspondentes aos três pronomes que significam "tu/você/vocês": **du** (informal singular), **ihr** (informal plural) e **Sie** (formal singular e plural).

Dominar o imperativo é essencial para a comunicação quotidiana, seja para pedir indicações, dar instruções ou simplesmente interagir com outras pessoas.

---

## 1. Visão Geral do Imperativo

### 1.1. As Três Formas

| Pronome | Uso | Forma | Exemplo |
|---------|-----|-------|---------|
| **du** | informal (1 pessoa) | Komm! | Vem! |
| **ihr** | informal (várias pessoas) | Kommt! | Vinde! |
| **Sie** | formal (1 ou várias pessoas) | Kommen Sie! | Venha! / Venham! |

### 1.2. Características Gerais

1. **Sem pronome** nas formas "du" e "ihr"
2. **Com pronome "Sie"** na forma formal (após o verbo)
3. **Ponto de exclamação** frequentemente usado
4. **Tom direto** (pode ser suavizado com "bitte")

---

## 2. Imperativo com DU (Informal Singular)

### 2.1. Formação Básica

**Regra geral:** Radical do verbo (sem terminação) + **(e)**

```
Infinitivo: kommen (vir)
Radical: komm-
Imperativo: Komm! ou Komme!
```

O "-e" final é opcional e geralmente omitido na fala coloquial:
- **Komm!** (mais comum)
- **Komme!** (mais formal/literário)

### 2.2. Verbos Regulares

| Infinitivo | Radical | Imperativo | Tradução |
|------------|---------|------------|----------|
| **machen** | mach- | Mach! | Faz! |
| **gehen** | geh- | Geh! | Vai! |
| **kommen** | komm- | Komm! | Vem! |
| **lernen** | lern- | Lern! | Aprende! |
| **spielen** | spiel- | Spiel! | Brinca!/Joga! |
| **arbeiten** | arbeit- | Arbeite! | Trabalha! |
| **warten** | wart- | Warte! | Espera! |
| **öffnen** | öffn- | Öffne! | Abre! |

**Observação:** Verbos que terminam em -ten, -den, -nen, -men geralmente mantêm o "-e":
```
Arbeite! (Trabalha!)
Warte! (Espera!)
Öffne! (Abre!)
Rechne! (Calcula!)
```

### 2.3. Verbos com Mudança Vocálica (e → i/ie)

Verbos que mudam "e" para "i" ou "ie" na 2ª e 3ª pessoa do presente mantêm essa mudança no imperativo, **mas sem o -e final**:

| Infinitivo | Presente (du) | Imperativo | Tradução |
|------------|---------------|------------|----------|
| **essen** | du isst | Iss! | Come! |
| **lesen** | du liest | Lies! | Lê! |
| **nehmen** | du nimmst | Nimm! | Pega!/Toma! |
| **sprechen** | du sprichst | Sprich! | Fala! |
| **sehen** | du siehst | Sieh! | Olha!/Vê! |
| **geben** | du gibst | Gib! | Dá! |
| **helfen** | du hilfst | Hilf! | Ajuda! |
| **treffen** | du triffst | Triff! | Encontra! |

**Exemplos:**
```
Iss das Brot! (Come o pão!)
Lies das Buch! (Lê o livro!)
Nimm einen Apfel! (Pega numa maçã!)
Sprich Deutsch! (Fala alemão!)
Sieh mal! (Olha!)
Gib mir das Buch! (Dá-me o livro!)
Hilf mir! (Ajuda-me!)
```

### 2.4. Verbos com Mudança Vocálica (a → ä) — NÃO mudam no imperativo

Verbos que mudam "a" para "ä" no presente **não** mudam no imperativo:

| Infinitivo | Presente (du) | Imperativo | Tradução |
|------------|---------------|------------|----------|
| **fahren** | du fährst | Fahr! | Conduz!/Vai! |
| **schlafen** | du schläfst | Schlaf! | Dorme! |
| **laufen** | du läufst | Lauf! | Corre! |
| **halten** | du hältst | Halt! | Para! |
| **tragen** | du trägst | Trag! | Usa!/Carrega! |

**Exemplos:**
```
Fahr langsam! (Conduz devagar!)
Schlaf gut! (Dorme bem!)
Lauf schnell! (Corre rápido!)
Halt! (Para!)
```

### 2.5. Verbos Irregulares Importantes

#### SEIN (ser/estar)
```
Imperativo: Sei!
Exemplos:
Sei ruhig! (Fica quieto!)
Sei vorsichtig! (Tem cuidado!)
Sei pünktlich! (Sê pontual!)
```

#### HABEN (ter)
```
Imperativo: Hab!
Exemplos:
Hab keine Angst! (Não tenhas medo!)
Hab Geduld! (Tem paciência!)
Hab Spaß! (Diverte-te!)
```

#### WERDEN (tornar-se)
```
Imperativo: Werd! / Werde!
Exemplos:
Werd(e) nicht nervös! (Não fiques nervoso!)
```

## 3. Imperativo com IHR (Informal Plural)

### 3.1. Formação

**Regra:** Usa a forma do presente de "ihr", mas **sem o pronome**.

```
Infinitivo: kommen
Presente: ihr kommt
Imperativo: Kommt!
```

**Esta é a forma mais fácil!**

### 3.2. Exemplos

| Infinitivo | Presente (ihr) | Imperativo | Tradução |
|------------|----------------|------------|----------|
| **machen** | ihr macht | Macht! | Fazei! |
| **gehen** | ihr geht | Geht! | Ide! |
| **kommen** | ihr kommt | Kommt! | Vinde! |
| **lernen** | ihr lernt | Lernt! | Aprendei! |
| **essen** | ihr esst | Esst! | Comei! |
| **lesen** | ihr lest | Lest! | Lede! |
| **nehmen** | ihr nehmt | Nehmt! | Pegai!/Tomai! |
| **sprechen** | ihr sprecht | Sprecht! | Falai! |
| **fahren** | ihr fahrt | Fahrt! | Conduzi!/Ide! |
| **schlafen** | ihr schlaft | Schlaft! | Dormi! |

**Exemplos em frases:**
```
Kommt her! (Vinde cá!)
Macht die Hausaufgaben! (Fazei os trabalhos de casa!)
Esst das Brot! (Comei o pão!)
Lest das Buch! (Lede o livro!)
Sprecht Deutsch! (Falai alemão!)
Fahrt vorsichtig! (Conduzi com cuidado!)
```

### 3.3. Verbos Irregulares

#### SEIN
```
Imperativo: Seid!
Exemplos:
Seid ruhig! (Ficai quietos!)
Seid vorsichtig! (Tende cuidado!)
Seid pünktlich! (Sede pontuais!)
```

#### HABEN
```
Imperativo: Habt!
Exemplos:
Habt keine Angst! (Não tenhais medo!)
Habt Geduld! (Tende paciência!)
Habt Spaß! (Divertam-se!)
```

## 4. Imperativo com SIE (Formal)

### 4.1. Formação

**Regra:** Usa a forma do presente de "Sie" + pronome "Sie" **após o verbo**.

```
Infinitivo: kommen
Presente: Sie kommen
Imperativo: Kommen Sie!
```

**Estrutura:**
```
Verbo + Sie + resto da frase!
```

### 4.2. Exemplos

| Infinitivo | Presente (Sie) | Imperativo | Tradução |
|------------|----------------|------------|----------|
| **machen** | Sie machen | Machen Sie! | Faça! |
| **gehen** | Sie gehen | Gehen Sie! | Vá! |
| **kommen** | Sie kommen | Kommen Sie! | Venha! |
| **lernen** | Sie lernen | Lernen Sie! | Aprenda! |
| **essen** | Sie essen | Essen Sie! | Coma! |
| **lesen** | Sie lesen | Lesen Sie! | Leia! |
| **nehmen** | Sie nehmen | Nehmen Sie! | Pegue!/Tome! |
| **sprechen** | Sie sprechen | Sprechen Sie! | Fale! |
| **fahren** | Sie fahren | Fahren Sie! | Conduza!/Vá! |
| **schlafen** | Sie schlafen | Schlafen Sie! | Durma! |

**Exemplos em frases:**
```
Kommen Sie herein! (Entre!)
Machen Sie die Tür zu! (Feche a porta!)
Essen Sie das Brot! (Coma o pão!)
Lesen Sie das Buch! (Leia o livro!)
Sprechen Sie Deutsch! (Fale alemão!)
Fahren Sie vorsichtig! (Conduza com cuidado!)
Nehmen Sie Platz! (Sente-se! / Tome assento!)
```

### 4.3. Verbos Irregulares

#### SEIN
```
Imperativo: Seien Sie!
Exemplos:
Seien Sie ruhig! (Fique quieto!)
Seien Sie vorsichtig! (Tenha cuidado!)
Seien Sie pünktlich! (Seja pontual!)
```

**Nota:** "Seien" é uma forma especial e irregular!

#### HABEN
```
Imperativo: Haben Sie!
Exemplos:
Haben Sie keine Angst! (Não tenha medo!)
Haben Sie Geduld! (Tenha paciência!)
Haben Sie einen schönen Tag! (Tenha um bom dia!)
```

## 5. Verbos Separáveis no Imperativo

### 5.1. Regra

O prefixo separa-se e vai para o final (como nas frases normais):

```
Infinitivo: aufstehen (levantar-se)

du: Steh auf!
ihr: Steht auf!
Sie: Stehen Sie auf!
```

### 5.2. Exemplos Comuns

| Infinitivo | du | ihr | Sie |
|------------|----|-----|-----|
| **aufstehen** | Steh auf! | Steht auf! | Stehen Sie auf! |
| **aufmachen** | Mach auf! | Macht auf! | Machen Sie auf! |
| **zumachen** | Mach zu! | Macht zu! | Machen Sie zu! |
| **mitkommen** | Komm mit! | Kommt mit! | Kommen Sie mit! |
| **mitmachen** | Mach mit! | Macht mit! | Machen Sie mit! |
| **anrufen** | Ruf an! | Ruft an! | Rufen Sie an! |
| **einsteigen** | Steig ein! | Steigt ein! | Steigen Sie ein! |
| **aussteigen** | Steig aus! | Steigt aus! | Steigen Sie aus! |
| **anfangen** | Fang an! | Fangt an! | Fangen Sie an! |
| **weitermachen** | Mach weiter! | Macht weiter! | Machen Sie weiter! |

**Exemplos em contexto:**
```
Steh auf! Es ist schon 8 Uhr! (Levanta-te! Já são 8 horas!)
Macht die Tür auf! (Abri a porta!)
Machen Sie das Fenster zu! (Feche a janela!)
Komm mit ins Kino! (Vem ao cinema!)
Ruft mich an! (Telefonai-me!)
Steigen Sie ein! (Entre! — no veículo)
Fang an! (Começa!)
```

## 6. Imperativo Negativo

### 6.1. Com NICHT

Para formar o imperativo negativo, adiciona-se **nicht**:

```
Komm! → Komm nicht!
Kommt! → Kommt nicht!
Kommen Sie! → Kommen Sie nicht!
```

**Exemplos:**
```
Geh nicht! (Não vás!)
Esst nicht zu viel! (Não comais demais!)
Sprechen Sie nicht so laut! (Não fale tão alto!)
Fahr nicht so schnell! (Não conduzas tão rápido!)
Schlaf nicht! (Não durmas!)
Macht das nicht! (Não façais isso!)
Seien Sie nicht nervös! (Não fique nervoso!)
```

### 6.2. Com KEIN

Quando há um substantivo, usa-se **kein** (nenhum):

```
Hab keine Angst! (Não tenhas medo!)
Macht keinen Lärm! (Não façais barulho!)
Haben Sie keine Sorge! (Não tenha preocupação!)
Esst kein Fast Food! (Não comais fast food!)
```

---

## 7. Suavizar o Imperativo com BITTE

### 7.1. Uso de "Bitte"

Para tornar o imperativo mais educado, adiciona-se **bitte** (por favor):

**Posições possíveis:**
1. No início: **Bitte** komm!
2. No meio: Komm **bitte**!
3. No fim: Komm, **bitte**!

**Exemplos:**
```
Bitte hilf mir! (Por favor, ajuda-me!)
Komm bitte her! (Vem cá, por favor!)
Machen Sie bitte die Tür zu! (Feche a porta, por favor!)
Wartet bitte! (Esperai, por favor!)
Geben Sie mir bitte das Buch! (Dê-me o livro, por favor!)
Ruf mich bitte an! (Telefona-me, por favor!)
```

### 7.2. Outras Formas de Suavizar

**MAL** (expressão coloquial, suaviza o tom):
```
Komm mal her! (Vem cá!)
Sieh mal! (Olha!)
Hör mal! (Ouve!)
Warte mal! (Espera um momento!)
Schau mal! (Olha!)
```

**DOCH** (encorajamento):
```
Komm doch! (Vá, vem!)
Iss doch! (Vá, come!)
Probier doch! (Vá, experimenta!)
```

**BITTE + MAL:**
```
Komm bitte mal her! (Vem cá, por favor!)
Hilf mir bitte mal! (Ajuda-me, por favor!)
```

## 9. Diálogos Práticos

### Diálogo 1: Em Casa (informal - du)
```
Mutter: Steh auf! Es ist schon 7 Uhr!
        (Levanta-te! Já são 7 horas!)

Kind: Noch fünf Minuten, bitte!
      (Mais cinco minutos, por favor!)

Mutter: Nein, komm jetzt! Mach dich fertig!
        (Não, vem agora! Prepara-te!)

Kind: Okay, okay. Wo ist mein T-Shirt?
      (Está bem, está bem. Onde está a minha t-shirt?)

Mutter: Schau im Schrank! Und beeil dich!
        (Olha no armário! E apressa-te!)
```

### Diálogo 2: Na Escola (informal - ihr)
```
Lehrer: Guten Morgen! Setzt euch, bitte!
        (Bom dia! Sentem-se, por favor!)

Schüler: Guten Morgen!
         (Bom dia!)

Lehrer: Öffnet eure Bücher auf Seite 25!
        (Abri os vossos livros na página 25!)

Lehrer: Lest den Text und macht dann die Übungen!
        (Lede o texto e depois fazei os exercícios!)

Lehrer: Arbeitet leise! Redet nicht!
        (Trabalhai em silêncio! Não faleis!)
```

### Diálogo 3: Pedindo Indicações (formal - Sie)
```
Tourist: Entschuldigung! Können Sie mir helfen?
         (Desculpe! Pode ajudar-me?)

Passant: Ja, natürlich!
         (Sim, claro!)

Tourist: Wie komme ich zum Bahnhof?
         (Como vou para o terminal?)

Passant: Gehen Sie geradeaus und biegen Sie dann links ab.
         (Vá em frente e depois vire à esquerda.)

Passant: Nehmen Sie die zweite Straße rechts.
         (Apanhe a segunda rua à direita.)

Tourist: Vielen Dank!
         (Muito obrigado!)

Passant: Bitte sehr! Viel Erfolg!
         (De nada! Muito sucesso!)
```

### Diálogo 4: No Restaurante (formal - Sie)
```
Kellner: Guten Abend! Nehmen Sie bitte Platz!
         (Boa noite! Sente-se, por favor!)

Gast: Danke schön!
      (Muito obrigado!)

Kellner: Hier ist die Speisekarte. Nehmen Sie sich Zeit!
         (Aqui está o menu. Tome o seu tempo!)

(Später)

Gast: Bringen Sie mir bitte ein Schnitzel!
      (Traga-me um schnitzel, por favor!)

Kellner: Sehr gern! Möchten Sie auch etwas trinken?
         (Com certeza! Gostaria também de beber algo?)

Gast: Ja, geben Sie mir bitte ein Bier!
      (Sim, dê-me uma cerveja, por favor!)
```

## 12. Erros Comuns e Como Evitá-los

### Erro 1: Usar pronome com "du" e "ihr"
**Incorreto:** *Du komm!* / *Ihr kommt!*
**Correto:** Komm! / Kommt!

### Erro 2: Esquecer "Sie" no imperativo formal
**Incorreto:** *Kommen!*
**Correto:** Kommen Sie!

### Erro 3: Manter o -e em verbos com mudança e→i
**Incorreto:** *Esse!* / *Liese!*
**Correto:** Iss! / Lies!

### Erro 4: Mudar a→ä no imperativo
**Incorreto:** *Fähr!*
**Correto:** Fahr!

### Erro 5: Não separar prefixos separáveis
**Incorreto:** *Aufsteh!*
**Correto:** Steh auf!

## 13. Resumo da Lição

### Pontos-Chave:

1. **Três formas de imperativo:**
   - **du:** Komm! (sem pronome)
   - **ihr:** Kommt! (sem pronome)
   - **Sie:** Kommen Sie! (com pronome)

2. **Verbos com e→i/ie:** Mantêm mudança, sem -e (Iss!, Lies!)

3. **Verbos com a→ä:** NÃO mudam (Fahr!, não Fähr!)

4. **Verbos separáveis:** Prefixo no final (Steh auf!)

5. **Negação:** nicht ou kein

6. **Suavizar:** bitte, mal, doch

7. **Irregulares importantes:**
   - sein: Sei! / Seid! / Seien Sie!
   - haben: Hab! / Habt! / Haben Sie!

## 14. Linguagem Formal, Informal e de Rua

O imperativo é uma das formas verbais onde mais se nota a diferença entre linguagem formal, informal e de rua. Vamos ver como os alemães realmente falam no dia a dia.

### Situação: Pedir para Alguém Vir

#### **Formal** (Sie)
```
Kommen Sie bitte herein!
Nehmen Sie bitte Platz!
```
**Tradução:** Entre, por favor! / Sente-se, por favor!

#### **Informal** (du)
```
Komm doch rein!
Setz dich hin!
```
**Tradução:** Vem cá dentro! / Senta-te!

#### **Rua/Jovem**
```
Komm mal rüber!
Setz dich, Alter!
```
**Tradução:** Vem cá! / Senta-te, mano!

**Observações:**
- **rein** = contração de "herein" (para dentro)
- **rüber** = contração de "herüber" (para cá)
- **Setz dich** = contração de "Setze dich" (senta-te)

### Situação: Pedir Silêncio

#### **Formal**
```
Seien Sie bitte ruhig!
Sprechen Sie bitte leiser!
```
**Tradução:** Fique quieto, por favor! / Fale mais baixo, por favor!

#### **Informal**
```
Sei ruhig!
Sprich leiser!
```
**Tradução:** Fica quieto! / Fala mais baixo!

#### **Rua/Jovem**
```
Halt die Klappe!
Halt's Maul! (muito rude)
Psst! Ruhe jetzt!
```
**Tradução:** Cala a boca! / Cala a boca! (muito rude) / Psst! Silêncio agora!

**Gírias e expressões:**
- **Halt die Klappe!** = Cala a boca! (coloquial, mas não extremamente rude)
- **Halt's Maul!** = Cala a boca! (muito rude, evitar)
- **Schnauze!** = Bico! (muito rude)
- **Sei still!** = Fica quieto! (neutro)

### Situação: Pedir Comida/Bebida

#### **Formal**
```
Geben Sie mir bitte ein Glas Wasser!
Bringen Sie mir bitte die Rechnung!
```
**Tradução:** Dê-me um copo de água, por favor! / Traga-me a conta, por favor!

#### **Informal**
```
Gib mir bitte ein Glas Wasser!
Bring mir bitte die Rechnung!
```
**Tradução:** Dá-me um copo de água, por favor! / Traz-me a conta, por favor!

#### **Rua/Jovem**
```
Gib mir mal 'n Wasser!
Hol mal die Rechnung!
```
**Tradução:** Dá-me uma água! / Vai buscar a conta!

**Contrações coloquiais:**
- **'n** = contração de "ein" (um)
- **mal** = partícula que suaviza o imperativo (já vimos antes)
- **Hol** = imperativo de "holen" (ir buscar)

### Situação: Dar Instruções/Ordens

#### **Formal**
```
Füllen Sie bitte dieses Formular aus!
Unterschreiben Sie hier, bitte!
```
**Tradução:** Preencha este formulário, por favor! / Assine aqui, por favor!

#### **Informal**
```
Füll bitte dieses Formular aus!
Unterschreib hier, bitte!
```
**Tradução:** Preenche este formulário, por favor! / Assina aqui, por favor!

#### **Rua/Jovem**
```
Mach mal das Formular fertig!
Unterschreib einfach hier!
```
**Tradução:** Acaba o formulário! / Assina simplesmente aqui!

### Situação: Pedir para Alguém Se Apressar

#### **Formal**
```
Beeilen Sie sich bitte!
Können Sie sich bitte beeilen?
```
**Tradução:** Apresse-se, por favor! / Pode se apressar, por favor?

#### **Informal**
```
Beeil dich!
Mach schnell!
```
**Tradução:** Apressa-te! / Despacha-te!

#### **Rua/Jovem**
```
Mach hin!
Los, los, los!
Beweg deinen Arsch! (rude)
Zack, zack!
```
**Tradução:** Despacha-te! / Vamos, vamos, vamos! / Mexe o cu! (rude) / Rápido, rápido!

**Expressões de rua:**
- **Mach hin!** = Despacha-te! (coloquial)
- **Los!** = Vamos! (usado para apressar)
- **Zack, zack!** = Rápido, rápido! (onomatopeia)
- **Beweg deinen Arsch!** = Mexe o cu! (muito coloquial/rude, evitar em contextos formais)

### Situação: Despedidas com Imperativo

#### **Formal**
```
Haben Sie einen schönen Tag!
Kommen Sie gut nach Hause!
Passen Sie auf sich auf!
```
**Tradução:** Tenha um bom dia! / Chegue bem a casa! / Cuide-se!

#### **Informal**
```
Hab einen schönen Tag!
Komm gut nach Hause!
Pass auf dich auf!
```
**Tradução:** Tem um bom dia! / Chega bem a casa! / Cuida-te!

#### **Rua/Jovem**
```
Mach's gut!
Bis dann!
Tschau! / Ciao!
Wir sehen uns!
```
**Tradução:** Fica bem! / Até logo! / Tchau! / A gente se vê!

**Observação:** "Mach's gut!" é a forma mais natural e comum de se despedir entre amigos (literalmente "Faz bem!").

### Tabela Comparativa de Expressões Comuns

| Formal | Informal | Rua/Jovem | Tradução |
|--------|----------|-----------|----------|
| Kommen Sie herein! | Komm rein! | Komm mal rüber! | Entre! / Vem cá! |
| Setzen Sie sich! | Setz dich! | Setz dich hin, Alter! | Sente-se! |
| Beeilen Sie sich! | Beeil dich! | Mach hin! | Apresse-se! |
| Seien Sie ruhig! | Sei ruhig! | Halt die Klappe! | Fique quieto! |

**Nota Cultural:** Jovens alemães usam muitas palavras emprestadas do inglês (chillen, checken, etc.) e contrações extremas. Também é comum usar "Alter" (literalmente "velho") como "mano" em português.
