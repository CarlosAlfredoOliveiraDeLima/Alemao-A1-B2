# Plural dos Substantivos - Padrões Mais Comuns

## Introdução

Formar o plural em alemão é outro desafio para estudantes, pois não existe uma única regra universal como em muitas outras línguas. Ao contrário do português, onde geralmente adicionamos -s ou -es, o alemão possui pelo menos cinco formas principais de formar o plural, além de alterações vocálicas chamadas Umlaut (ä, ö, ü).

A boa notícia é que existem padrões e tendências que podem ajudar. Além disso, no plural, **todos os substantivos usam o artigo "die"**, independentemente do género no singular.

Nesta lição, vamos explorar sistematicamente os padrões de formação do plural e estratégias para memorizá-los.

---

## 1. Visão Geral do Plural Alemão

### 1.1. O Artigo no Plural

**Regra fundamental:** Independentemente do género no singular (der, die, das), o artigo no plural é sempre **die**.

```
Singular → Plural

der Tisch → die Tische (a mesa → as mesas)
die Lampe → die Lampen (o candeeiro → os candeeiros)
das Buch → die Bücher (o livro → os livros)
```

### 1.2. Os Cinco Padrões Principais

| Padrão | Mudança | Exemplo Singular | Exemplo Plural |
|--------|---------|------------------|----------------|
| **Tipo 1** | sem mudança (-) | der Lehrer | die Lehrer |
| **Tipo 2** | Umlaut (¨) | der Vater | die Väter |
| **Tipo 3** | -e | der Tisch | die Tische |
| **Tipo 4** | Umlaut + -e | die Hand | die Hände |
| **Tipo 5** | -er | das Kind | die Kinder |
| **Tipo 6** | Umlaut + -er | das Buch | die Bücher |
| **Tipo 7** | -n | die Lampe | die Lampen |
| **Tipo 8** | -en | die Frau | die Frauen |
| **Tipo 9** | -s | das Auto | die Autos |

**Observação:** O Umlaut (¨) transforma:
- a → ä
- o → ö
- u → ü
- au → äu

## 2. Padrão 1: Sem Mudança (-)

### 2.1. Descrição

O substantivo permanece exatamente igual no plural. Apenas o artigo muda de der/das para die.

### 2.2. Quando Ocorre

**A. Substantivos masculinos terminados em -er, -el, -en:**

```
der Lehrer → die Lehrer (professor/professores)
der Schüler → die Schüler (aluno/alunos)
der Fahrer → die Fahrer (motorista/motoristas)
der Fehler → die Fehler (erro/erros)
der Computer → die Computer (computador/computadores)

der Löffel → die Löffel (colher/colheres)
der Schlüssel → die Schlüssel (chave/chaves)
der Mantel → die Mantel (casaco/casacos)
der Onkel → die Onkel (tio/tios)

der Wagen → die Wagen (carro/carros)
der Kuchen → die Kuchen (bolo/bolos)
der Garten → die Garten (jardim/jardins)
```

**B. Substantivos neutros terminados em -er, -el, -en, -chen, -lein:**

```
das Fenster → die Fenster (janela/janelas)
das Zimmer → die Zimmer (quarto/quartos)
das Messer → die Messer (faca/facas)

das Mittel → die Mittel (meio/meios)
das Siegel → die Siegel (selo/selos)

das Zeichen → die Zeichen (sinal/sinais)
das Kissen → die Kissen (almofada/almofadas)

das Mädchen → die Mädchen (rapariga/raparigas)
das Brötchen → die Brötchen (pãozinho/pãezinhos)
das Häuschen → die Häuschen (casinha/casinhas)
```

### 2.3. Exemplos em Frases

```
Der Lehrer ist gut. → Die Lehrer sind gut.
(O professor é bom. → Os professores são bons.)

Das Fenster ist offen. → Die Fenster sind offen.
(A janela está aberta. → As janelas estão abertas.)

Ich habe einen Computer. → Ich habe zwei Computer.
(Tenho um computador. → Tenho dois computadores.)

Das Mädchen spielt. → Die Mädchen spielen.
(A rapariga brinca. → As raparigas brincam.)
```

## 3. Padrão 2: Apenas Umlaut (¨)

### 3.1. Descrição

O substantivo recebe Umlaut na vogal tónica, mas não adiciona terminação.

### 3.2. Quando Ocorre

**Alguns substantivos masculinos (não há regra fixa, deve memorizar):**

```
der Vater → die Väter (pai/pais)
der Bruder → die Brüder (irmão/irmãos)
der Apfel → die Äpfel (maçã/maçãs)
der Mantel → die Mäntel (casaco/casacos)
der Vogel → die Vögel (pássaro/pássaros)
der Garten → die Gärten (jardim/jardins)
der Hafen → die Häfen (porto/portos)
```

**Nota:** Compare com o Padrão 1. Algumas palavras com -er/-el/-en não recebem Umlaut (der Fehler → die Fehler), enquanto outras recebem (der Vater → die Väter). Deve memorizar cada caso.

### 3.3. Exemplos em Frases

```
Mein Vater arbeitet. → Meine Väter arbeiten.
(O meu pai trabalha. → Os meus pais trabalham.)
[Nota: "Väter" aqui significa "pais" no sentido de pai+pai ou antepassados]

Der Apfel ist rot. → Die Äpfel sind rot.
(A maçã é vermelha. → As maçãs são vermelhas.)

Ein Vogel singt. → Viele Vögel singen.
(Um pássaro canta. → Muitos pássaros cantam.)
```

## 4. Padrão 3: Adicionar -e

### 4.1. Descrição

Adiciona-se a terminação **-e** ao substantivo. Sem Umlaut.

### 4.2. Quando Ocorre

**A. Muitos substantivos masculinos de uma sílaba:**

```
der Tisch → die Tische (mesa/mesas)
der Tag → die Tage (dia/dias)
der Hund → die Hunde (cão/cães)
der Freund → die Freunde (amigo/amigos)
der Platz → die Plätze (lugar/lugares) — *exceção: tem Umlaut*
der Punkt → die Punkte (ponto/pontos)
der Stift → die Stifte (caneta/canetas)
der Monat → die Monate (mês/meses)
der Beruf → die Berufe (profissão/profissões)
der Schuh → die Schuhe (sapato/sapatos)
```

**B. Alguns substantivos neutros:**

```
das Brot → die Brote (pão/pães)
das Heft → die Hefte (caderno/cadernos)
das Jahr → die Jahre (ano/anos)
das Tier → die Tiere (animal/animais)
das Beispiel → die Beispiele (exemplo/exemplos)
das Problem → die Probleme (problema/problemas)
```

**C. Alguns substantivos femininos (poucos):**

```
die Hand → die Hände (mão/mãos) — *com Umlaut, veremos no próximo padrão*
```

### 4.3. Exemplos em Frases

```
Der Tisch ist neu. → Die Tische sind neu.
(A mesa é nova. → As mesas são novas.)

Ein Tag hat 24 Stunden. → Sieben Tage hat eine Woche.
(Um dia tem 24 horas. → Sete dias tem uma semana.)

Ich habe einen Hund. → Wir haben drei Hunde.
(Tenho um cão. → Temos três cães.)

Das Problem ist schwer. → Die Probleme sind schwer.
(O problema é difícil. → Os problemas são difíceis.)
```

## 5. Padrão 4: Umlaut + -e

### 5.1. Descrição

Adiciona-se **-e** E a vogal recebe Umlaut (¨).

### 5.2. Quando Ocorre

**A. Muitos substantivos masculinos de uma sílaba com a, o, u:**

```
der Sohn → die Söhne (filho/filhos)
der Stuhl → die Stühle (cadeira/cadeiras)
der Zug → die Züge (comboio/comboios)
der Ball → die Bälle (bola/bolas)
der Baum → die Bäume (árvore/árvores)
der Platz → die Plätze (lugar/lugares)
der Satz → die Sätze (frase/frases)
der Arzt → die Ärzte (médico/médicos)
der Gast → die Gäste (hóspede/hóspedes)
der Kopf → die Köpfe (cabeça/cabeças)
```

**B. Alguns substantivos femininos com a, o, u:**

```
die Hand → die Hände (mão/mãos)
die Stadt → die Städte (cidade/cidades)
die Wand → die Wände (parede/paredes)
die Nacht → die Nächte (noite/noites)
die Kraft → die Kräfte (força/forças)
die Kuh → die Kühe (vaca/vacas)
die Wurst → die Würste (salsicha/salsichas)
die Maus → die Mäuse (rato/ratos)
```

### 5.3. Exemplos em Frases

```
Der Stuhl ist bequem. → Die Stühle sind bequem.
(A cadeira é confortável. → As cadeiras são confortáveis.)

Mein Sohn lernt Deutsch. → Meine Söhne lernen Deutsch.
(O meu filho aprende alemão. → Os meus filhos aprendem alemão.)

Die Stadt ist groß. → Die Städte sind groß.
(A cidade é grande. → As cidades são grandes.)

Eine Hand hat fünf Finger. → Zwei Hände haben zehn Finger.
(Uma mão tem cinco dedos. → Duas mãos têm dez dedos.)
```

## 6. Padrão 5: Adicionar -er

### 6.1. Descrição

Adiciona-se a terminação **-er**. Este padrão é quase exclusivo dos substantivos neutros.

### 6.2. Quando Ocorre

**A. Muitos substantivos neutros de uma sílaba:**

```
das Kind → die Kinder (criança/crianças)
das Bild → die Bilder (imagem/imagens)
das Ei → die Eier (ovo/ovos)
das Feld → die Felder (campo/campos)
das Lied → die Lieder (canção/canções)
das Kleid → die Kleider (vestido/vestidos)
das Schild → die Schilder (placa/placas)
das Rad → die Räder (roda/rodas) — *com Umlaut*
```

**B. Alguns substantivos masculinos (raros):**

```
der Mann → die Männer (homem/homens) — *com Umlaut*
der Wald → die Wälder (floresta/florestas) — *com Umlaut*
der Geist → die Geister (espírito/espíritos)
der Rand → die Ränder (margem/margens) — *com Umlaut*
```

### 6.3. Exemplos em Frases

```
Das Kind spielt. → Die Kinder spielen.
(A criança brinca. → As crianças brincam.)

Ein Bild hängt an der Wand. → Viele Bilder hängen an der Wand.
(Uma imagem está pendurada na parede. → Muitas imagens estão penduradas na parede.)

Der Mann ist groß. → Die Männer sind groß.
(O homem é grande. → Os homens são grandes.)

Ich esse ein Ei. → Ich esse zwei Eier.
(Como um ovo. → Como dois ovos.)
```

## 7. Padrão 6: Umlaut + -er

### 7.1. Descrição

Adiciona-se **-er** E a vogal recebe Umlaut. Muito comum com substantivos neutros.

### 7.2. Quando Ocorre

**A. Muitos substantivos neutros com a, o, u:**

```
das Buch → die Bücher (livro/livros)
das Haus → die Häuser (casa/casas)
das Dach → die Dächer (telhado/telhados)
das Fach → die Fächer (disciplina/disciplinas)
das Glas → die Gläser (copo/copos)
das Land → die Länder (país/países)
das Tuch → die Tücher (pano/panos)
das Wort → die Wörter (palavra/palavras — quando significam palavras isoladas)
```

**B. Alguns substantivos masculinos:**

```
der Mann → die Männer (homem/homens)
der Wald → die Wälder (floresta/florestas)
der Rand → die Ränder (margem/margens)
```

### 7.3. Exemplos em Frases

```
Das Buch ist interessant. → Die Bücher sind interessant.
(O livro é interessante. → Os livros são interessantes.)

Ein Haus kostet viel. → Häuser kosten viel.
(Uma casa custa muito. → Casas custam muito.)

Das Land ist schön. → Die Länder sind schön.
(O país é bonito. → Os países são bonitos.)

Ich trinke aus einem Glas. → Wir trinken aus drei Gläsern.
(Bebo de um copo. → Bebemos de três copos.)
```

## 8. Padrão 7: Adicionar -n

### 8.1. Descrição

Adiciona-se apenas **-n**. Sem Umlaut.

### 8.2. Quando Ocorre

**Substantivos femininos terminados em -e:**

```
die Lampe → die Lampen (candeeiro/candeeiros)
die Blume → die Blumen (flor/flores)
die Straße → die Straßen (rua/ruas)
die Schule → die Schulen (escola/escolas)
die Tasche → die Taschen (bolsa/bolsas)
die Frage → die Fragen (pergunta/perguntas)
die Seite → die Seiten (página/páginas)
die Minute → die Minuten (minuto/minutos)
die Stunde → die Stunden (hora/horas)
die Woche → die Wochen (semana/semanas)
die Farbe → die Farben (cor/cores)
die Stimme → die Stimmen (voz/vozes)
```

**Regra prática:** Quase todos os substantivos femininos terminados em -e formam o plural em -n.

### 8.3. Exemplos em Frases

```
Die Lampe ist hell. → Die Lampen sind hell.
(O candeeiro é luminoso. → Os candeeiros são luminosos.)

Eine Blume ist schön. → Viele Blumen sind schön.
(Uma flor é bonita. → Muitas flores são bonitas.)

Ich habe eine Frage. → Ich habe viele Fragen.
(Tenho uma pergunta. → Tenho muitas perguntas.)

Eine Stunde hat 60 Minuten. → Zwei Stunden haben 120 Minuten.
(Uma hora tem 60 minutos. → Duas horas têm 120 minutos.)
```

## 9. Padrão 8: Adicionar -en

### 9.1. Descrição

Adiciona-se **-en**. Sem Umlaut. Este é o padrão mais produtivo em alemão.

### 9.2. Quando Ocorre

**A. Maioria dos substantivos femininos (especialmente os que não terminam em -e):**

```
die Frau → die Frauen (mulher/mulheres)
die Tür → die Türen (porta/portas)
die Zahl → die Zahlen (número/números)
die Zeit → die Zeiten (tempo/tempos)
die Uhr → die Uhren (relógio/relógios)
die Antwort → die Antworten (resposta/respostas)
die Arbeit → die Arbeiten (trabalho/trabalhos)
```

**B. Todos os substantivos femininos com terminações abstratas:**

```
die Übung → die Übungen (exercício/exercícios)
die Wohnung → die Wohnungen (apartamento/apartamentos)
die Zeitung → die Zeitungen (jornal/jornais)
die Lösung → die Lösungen (solução/soluções)

die Freiheit → die Freiheiten (liberdade/liberdades)
die Möglichkeit → die Möglichkeiten (possibilidade/possibilidades)

die Freundschaft → die Freundschaften (amizade/amizades)
die Wissenschaft → die Wissenschaften (ciência/ciências)

die Information → die Informationen (informação/informações)
die Nation → die Nationen (nação/nações)
```

**C. Substantivos masculinos "fracos" (veremos mais na lição 9):**

```
der Student → die Studenten (estudante/estudantes)
der Mensch → die Menschen (pessoa/pessoas)
der Herr → die Herren (senhor/senhores)
der Junge → die Jungen (rapaz/rapazes)
der Kunde → die Kunden (cliente/clientes)
```

**D. Alguns substantivos neutros:**

```
das Auge → die Augen (olho/olhos)
das Ohr → die Ohren (orelha/orelhas)
das Bett → die Betten (cama/camas)
das Hemd → die Hemden (camisa/camisas)
das Ende → die Enden (fim/fins)
das Interesse → die Interessen (interesse/interesses)
```

### 9.3. Exemplos em Frases

```
Die Frau arbeitet. → Die Frauen arbeiten.
(A mulher trabalha. → As mulheres trabalham.)

Eine Tür ist geschlossen. → Alle Türen sind geschlossen.
(Uma porta está fechada. → Todas as portas estão fechadas.)

Der Student lernt. → Die Studenten lernen.
(O estudante aprende. → Os estudantes aprendem.)

Das Auge ist wichtig. → Die Augen sind wichtig.
(O olho é importante. → Os olhos são importantes.)
```

## 10. Padrão 9: Adicionar -s

### 10.1. Descrição

Adiciona-se **-s** (como em inglês ou português). Este é o padrão para palavras estrangeiras e algumas abreviaturas.

### 10.2. Quando Ocorre

**A. Palavras estrangeiras (especialmente de origem inglesa ou francesa):**

```
das Auto → die Autos (carro/carros)
das Kino → die Kinos (cinema/cinemas)
das Hotel → die Hotels (hotel/hotéis)
das Café → die Cafés (café/cafés)
das Restaurant → die Restaurants (restaurante/restaurantes)
das Büro → die Büros (escritório/escritórios)
das Hobby → die Hobbys (passatempo/passatempos)
das Video → die Videos (vídeo/vídeos)
das Taxi → die Taxis (táxi/táxis)
die Pizza → die Pizzas (pizza/pizzas)
der Park → die Parks (parque/parques)
der Test → die Tests (teste/testes)
```

**B. Abreviaturas:**

```
der Lkw → die Lkws (camião/camiões)
der Pkw → die Pkws (carro de passageiros/carros)
die CD → die CDs (CD/CDs)
die DVD → die DVDs (DVD/DVDs)
```

**C. Nomes próprios:**

```
die Müllers (os Müller — a família Müller)
die Schmidts (os Schmidt)
```

### 10.3. Exemplos em Frases

```
Das Auto ist schnell. → Die Autos sind schnell.
(O carro é rápido. → Os carros são rápidos.)

Ein Kino zeigt Filme. → Viele Kinos zeigen Filme.
(Um cinema mostra filmes. → Muitos cinemas mostram filmes.)

Ich habe ein Hobby. → Wir haben viele Hobbys.
(Tenho um passatempo. → Temos muitos passatempos.)
```

## 11. Casos Especiais e Exceções

### 11.1. Substantivos que Não Mudam

Alguns substantivos são idênticos no singular e plural (como "Lehrer"):

```
der Lehrer → die Lehrer
das Fenster → die Fenster
der Schüler → die Schüler
das Mädchen → die Mädchen
```

### 11.2. Dois Plurais Diferentes com Significados Diferentes

**das Wort:**
```
das Wort → die Wörter (palavras isoladas, vocabulário)
das Wort → die Worte (palavras em contexto, ditos, expressões)

Exemplo:
- Ich lerne neue Wörter. (Aprendo palavras novas — vocabulário.)
- Er spricht weise Worte. (Ele diz palavras sábias — expressões.)
```

**die Bank:**
```
die Bank → die Banken (bancos — instituições financeiras)
die Bank → die Bänke (bancos — para sentar)

Exemplo:
- Die Banken sind geschlossen. (Os bancos estão fechados — instituições.)
- Die Bänke im Park sind neu. (Os bancos no parque são novos — assentos.)
```

**der Rat:**
```
der Rat → die Räte (conselhos — órgãos)
der Rat → die Ratschläge (conselhos — sugestões)
```

### 11.3. Substantivos que Existem Apenas no Singular

**Substantivos abstratos ou materiais:**
```
die Milch (o leite)
das Wasser (a água)
der Reis (o arroz)
die Musik (a música)
die Liebe (o amor)
das Glück (a felicidade)
```

**Nota:** Se quiser expressar pluralidade, usa-se outras palavras:
- zwei Flaschen Milch (duas garrafas de leite)
- verschiedene Arten von Musik (vários tipos de música)

### 11.4. Substantivos que Existem Apenas no Plural

```
die Eltern (os pais)
die Leute (as pessoas)
die Ferien (as férias)
die Lebensmittel (os alimentos)
die Kosten (os custos)
```

**Exemplos:**
- Meine Eltern wohnen in Berlin. (Os meus pais vivem em Berlim.)
- Viele Leute sind hier. (Muitas pessoas estão aqui.)
- Die Ferien sind im Sommer. (As férias são no verão.)

## 12. Estratégias de Memorização

### 12.1. Sempre Aprender com Três Formas

Ao memorizar um substantivo, aprenda sempre:
1. **Artigo** (género)
2. **Singular**
3. **Plural**

```
der Tisch, die Tische
die Lampe, die Lampen
das Buch, die Bücher
```

Ou use a notação comum em dicionários:
```
Tisch, der, -e
Lampe, die, -n
Buch, das, ¨-er (onde ¨ indica Umlaut)
```

### 12.2. Agrupe por Padrão

Organize seu vocabulário por padrão de plural:

**Grupo -er/-¨er:**
```
das Kind → die Kinder
das Buch → die Bücher
das Haus → die Häuser
```

**Grupo -e/-¨e:**
```
der Tisch → die Tische
der Stuhl → die Stühle
die Stadt → die Städte
```

**Grupo -n/-en:**
```
die Lampe → die Lampen
die Frau → die Frauen
der Student → die Studenten
```

### 12.3. Use Flashcards com Contexto

Em vez de memorizar apenas a palavra:
```
Frente: der Tisch
Verso: die Tische
```

Use frases:
```
Frente: Der Tisch ist groß.
Verso: Die Tische sind groß.
```

### 12.4. Preste Atenção às Tendências

**Femininos geralmente fazem plural em -n/-en:**
- die Frau → die Frauen
- die Lampe → die Lampen
- die Übung → die Übungen

**Neutros frequentemente fazem plural em -er/-¨er:**
- das Kind → die Kinder
- das Buch → die Bücher
- das Haus → die Häuser

**Masculinos são mais variados:**
- der Tisch → die Tische (-e)
- der Stuhl → die Stühle (-¨e)
- der Lehrer → die Lehrer (sem mudança)
- der Vater → die Väter (apenas Umlaut)

### 12.5. Pratique com Números

Use números ao praticar o plural:
```
ein Buch → zwei Bücher → drei Bücher
eine Lampe → zwei Lampen → drei Lampen
ein Kind → zwei Kinder → drei Kinder
```

## 15. Erros Comuns e Como Evitá-los

### Erro 1: Adicionar -s a todas as palavras
**Incorreto:** *die Buchs, die Tisches*
**Correto:** die Bücher, die Tische

**Solução:** O -s só se usa em palavras estrangeiras. Palavras alemãs têm outros padrões.

### Erro 2: Esquecer o Umlaut
**Incorreto:** *die Bucher* (sem Umlaut)
**Correto:** die Bücher

**Solução:** Preste atenção ao Umlaut. Ele muda a pronúncia e é essencial.

### Erro 3: Usar o artigo errado no plural
**Incorreto:** *das Kinder*
**Correto:** die Kinder

**Solução:** TODOS os plurais usam "die", independentemente do género no singular.

### Erro 4: Não memorizar o plural junto com o singular
**Solução:** Sempre aprenda as três formas: artigo, singular, plural.

### Erro 5: Confundir singular e plural idênticos
**Problema:** "der Lehrer" pode ser singular ou plural.
**Solução:** Preste atenção ao contexto e ao verbo:
- Der Lehrer arbeitet. (singular)
- Die Lehrer arbeiten. (plural)

| Padrão | Mudança | Género Comum | Exemplos |
|--------|---------|--------------|----------|
| **1** | - (sem mudança) | M, N | der Lehrer → die Lehrer |
| **2** | ¨ (apenas Umlaut) | M | der Vater → die Väter |
| **3** | -e | M, N | der Tisch → die Tische |
| **4** | ¨-e (Umlaut + e) | M, F | der Stuhl → die Stühle |
| **5** | -er | N, M (raro) | das Kind → die Kinder |
| **6** | ¨-er (Umlaut + er) | N, M (raro) | das Buch → die Bücher |
| **7** | -n | F (com -e) | die Lampe → die Lampen |
| **8** | -en | F, M (fracos), N | die Frau → die Frauen |
| **9** | -s | estrangeiras | das Auto → die Autos |

## 17. Resumo da Lição

### Pontos-Chave:

1. **Todos os plurais usam "die"** como artigo, independentemente do género no singular

2. **Nove padrões principais** de formação do plural, com variações de Umlaut

3. **Femininos** geralmente fazem plural em -n ou -en

4. **Neutros** frequentemente fazem plural em -er ou -¨er

5. **Masculinos** são mais variados, mas muitos fazem -e ou -¨e

6. **Palavras estrangeiras** geralmente adicionam -s

7. **Sempre memorizar** artigo + singular + plural juntos

8. **Umlaut** (ä, ö, ü) é essencial e muda a pronúncia
